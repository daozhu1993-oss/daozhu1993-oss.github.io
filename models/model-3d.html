3D 赋能模型

3D Empowerment
你越盯，下属越不动脑？给团队装上三颗自主的轮子。
蒋小华  ·  《赋能工作法》  ·  自决定 · 自发展 · 自驱动
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 3D 赋能模型——自决定、自发展、自驱动，给团队装上三颗自主的轮子。企业未来拼的不是谁管得狠，而是谁赋能得好；你替他想，他就懒得想。
Origin
蒋小华《赋能工作法》中提出，面向团队赋能的领导力框架。
Field
领导力 / 团队管理管得越细团队越木、下属不动脑时，用它换思路。
Core Definition
三个自 D自决定 Decide · 自发展 Develop · 自驱动 Drive。
Thinking Layer
领导层 / 授权赋能在「控制」之外，学会让团队自己跑。

控制式管理 vs 3D 赋能

Bad vs Good
✕

把管理当成控制

Control Mode
  • 事无巨细盯进度，团队像提线木偶
  • 替团队做决定，他自然懒得想
  • 管得越细，团队越不动脑
  • 你这根鞭子一抽，他才动一下
例：一开始事无巨细盯进度，成员天天等指令，没人主动。
✓

3D 赋能

Empower to Run
  • D1 自决定：让团队自己拿主意
  • D2 自发展：陪跑成就人，让他长本事
  • D3 自驱动：拔掉鞭子也照样往前冲
  • 用目标 + 边界替代盯过程
例：把决策权交出去，只定目标和边界，一个月后周报自己就来了。

3D 三颗轮子

The Three Ds
D1

自决定 Self-Decide

让团队自己拿主意。控制欲越强，团队越弱——你替他想，他就懒得想。

D2

自发展 Self-Develop

和人才共同成就，让人获得自我成长。赋能不是放羊，是陪跑，让他在做事中长本事。

D3

自驱动 Self-Drive

让团队主动奔跑。真正的好团队，拔掉你这根「鞭子」也照样往前冲。

落地五步

Light the Fire

列「替团队做的决定」

挑一半还回去，让他自己拿，别抢方向盘。

交目标 + 边界

只定方向和不做什么，过程交给他。

把「培养人」写进目标

而不只是「完成事」——让他长本事。

退后当陪跑

不盯过程，在他卡点时给支持。

让自驱动点燃

不用催，周报自己来，管理成本直线降。
