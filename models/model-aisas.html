AISAS 模型

AISAS Model
投了广告用户却划走？搜完信了、买了还愿说，才算真到手。
电通  ·  Dentsu  ·  2005 年消费者行为模型
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： AISAS 把消费者路径重写了一遍——注意、兴趣、搜索、行动、分享。搜索和分享这两步，是互联网给旧漏斗加的新引擎；用户不是看了就买，是搜完信了、买了还愿意说，才算真的到手。
Origin
电通（Dentsu）2005 年提出，重构互联网时代的消费者行为路径。
Field
消费者行为 / 营销投了广告没转化、种草没人搜时，用它看全链路。
Core Definition
五步路径Attention 注意 · Interest 兴趣 · Search 搜索 · Action 行动 · Share 分享。
Thinking Layer
路径层 / 用户旅程在「做曝光」之前，先把从看到买的全链路想通。

旧漏斗梦 vs AISAS

Bad vs Good
✕

活在「看了就买」的旧梦

Buy-on-Sight Myth
  • 只买曝光，砸信息流拉满 A-I
  • 不做口碑，一搜全是「踩雷」直接劝退
  • 买完没人分享，白白错过免费传播
  • 钱花在入口，转化卡在搜索这关
例：护肤品牌砸信息流（A-I 拉满），可小红书一搜全是踩雷，搜索关直接劝退。
✓

AISAS 全链路经营

Full-Loop Marketing
  • 前半段 A-I 做流量入口，先有钩子被看见
  • 中间 S 当转化主战场，口碑才是真开关
  • 后半段 A-S，买完引导分享做免费传播
  • 投放和口碑一起经营，顺序不能乱
例：先铺真实测评、再做投放，转化翻了倍——顺序对了，钱才不白花。

AISAS 五步拆解

The Five Steps
A

注意 Attention

流量入口。没钩子连被看见的资格都没有。

I

兴趣 Interest

勾住人。让刷到的人愿意多停留一秒。

S

搜索 Search（新引擎）

用户不信广告，只信自己搜出来的评价；口碑才是真正的转化开关。

A

行动 Action

下单。但买完不算完——还有最后一步。

S

分享 Share（新引擎）

买完不发朋友圈，等于白做一次免费传播。

落地五步

Run the Loop

先铺口碑

别只买曝光，先确保「一搜你」出来的是好话。

入口做钩子

A-I 拉满，让用户有资格被看见、愿意停留。

搜索当主战场

口碑和投放一起经营，别让评价劝退。

埋分享钩子

每个触点都埋一个，让买的人替你说话。

复盘链路顺序

检查「看了→搜信→买→说」是不是全通了。

模型金句

In a Nutshell

用户不是看了就买，是搜完信了、买了还愿意说，才算真的到手。

AISAS 的本质

用户不信你广告，只信自己搜出来的评价。

搜索关的玄机

分享关的价值

马上能用的事

Put It to Work
✓

看评价干不干净，一搜全是踩雷就先别投。

✓

把搜索当主战场

口碑和投放一起做，别只追曝光量。

✓

每个触点埋分享钩子

让买的人替你说话，免费传播才发生。

✓

找 KOC 铺真实测评

比硬广更镇搜索关，评价才是转化开关。

✓

盯「搜→信→买」转化

别只报曝光，链路通了才算数。

✓

复盘顺序对不对

A-I-S-A-S 通了吗？顺序乱，钱就打水漂。

什么时候该掏出它

When to Use
营销投放 品牌种草 消费者行为 流量运营 口碑管理 小红书 / 抖音 转化优化 任何「投了广告却划走」的时候
