布鲁姆分类法

Bloom's Taxonomy
本杰明·布鲁姆  ·  Benjamin Bloom  ·  教育目标分类学 · 认知型
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 布鲁姆分类法把学习从低到高分成六个层次——记忆、理解、应用、分析、评价、创造。它是一张诊断你「学到哪了」的 CT 扫描图：多数人卡在前两层，而人与人的差距，恰恰在高阶那几层。
Origin
本杰明·布鲁姆（Benjamin Bloom）1956 年提出教育目标分类学，2001 年修订为如今六层。
Field
学习诊断 / 认知层级 / 教学设计学生、老师、自学者都能用来给自己"照 CT"。
Core Definition
六层认知阶梯记忆 → 理解 → 应用 → 分析 → 评价 → 创造，越往上越深。
Thinking Layer
认知诊断层在"学没学"之后，专门解决"学到第几层"。

停在低阶 vs 爬向高阶

Know vs Change
✕

停在低阶（知道）

Stuck at the Bottom
  • 背得滚瓜烂熟，考完三天全忘光
  • "我懂了"是最大的幻觉陷阱
  • 学了很多，真要用时拿不出来
  • 用勤奋掩盖懒惰，一直在底层空转
例：读完一本书，别人问讲了什么，你只能蹦出几个关键词。
✓

爬向高阶（改变）

Climb to the Top
  • 主动调用知识，解决真实问题
  • 看透论证漏洞与隐藏假设
  • 形成有依据的独立判断
  • 创造新东西，从消费者变生产者
例：学完 10 个模型，设计出一套自己的问题解决流程——这是第六层的"创造"。

六层认知阶梯

Six Levels of Cognition
L1

记忆

从大脑调取信息，能复述、识别、背出。考试前背得滚瓜烂熟，考完三天全忘——这是起点，绝不是终点。

L2

理解

用自己的话解释概念、概括大意、举出例子。注意："我懂了"往往是幻觉最大的陷阱。

自检：你能把这个概念讲给一个完全不懂的人听吗？
L3

应用

把知识用到新的具体情境。学完沟通模型，下次吵架真的用上了——这是从"知道"到"做到"的关键一跃。

L4

分析

把复杂信息拆开，看清各部分关系、逻辑与隐含假设。能看出作者的论证漏洞——批判性思维的起点。

L5

评价

基于标准，对观点、方法、作品做出有依据的判断。能说清"为什么这个比那个好"，而不只是"我喜欢"。

L6

创造

把所学要素重新组合，产出新观点、新方案、新作品。这是学习的终极形态：从消费者变成生产者。

例：基于多个模型，设计一套你自己的问题解决流程。

向上爬一层的五步

Climb One Level at a Time

自我诊断

拿一件最近"学过"的东西，诚实问自己：我目前停留在哪一层？

理解→应用

别急着学新东西，先找三个生活场景，把现有知识刻意用一次。

应用→分析

对比这个知识和另一个类似知识，找出异同与背后的隐藏假设。

分析→评价

写一篇短文，有理有据地评价它的价值——逼自己形成独立判断。

评价→创造

基于它，设计一个你自己的东西：文章、方案、分享都算创造。

模型金句

In a Nutshell

低阶学习让你"知道"，高阶学习让你"改变"。

学习的两种结局

只停留在金字塔底层的学习，本质上是在用勤奋掩盖懒惰。

岛主批注

真正的学霸，不是比你多背了多少，而是在同样的内容上，比你多爬了几层。

Put It to Work
✓

学完先诊断在第几层

记忆/理解/应用/分析/评价/创造，先定位，再向上爬。

✓

把"我懂了"换成"我能讲"

讲给外行听一遍，讲不通就是你还在第二层。

✓

找 3 个场景用一次

从理解跨到应用，是"知道"变"做到"的关键一跃。

✓

读文章时找隐藏假设

练分析层：作者没说出口的前提，往往才是关键。

✓

就一个观点写评价

有理有据地说"为什么好/不好"，逼出独立判断。

✓

基于所学创造点什么

写文章、做方案、设计流程——创造是学习的最顶端。
