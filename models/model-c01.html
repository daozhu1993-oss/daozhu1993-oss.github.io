人货场

People · Goods · Place
零售就三件事——谁买（人）、卖什么（货）、在哪买（场）；它是一张旋转门，核心角一直在变。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 人货场不是一份静态的三要素清单，而是「谁买、卖什么、在哪买」组成的旋转门——核心角从「场」转到「货」再转到「人」。你的困境，多半是用上一个时代的核心角，打当下的仗。
Origin
岛主解读 / 体系零售与新零售的经典三要素框架，岛主做了动态化重读。
Field
零售 / 电商 / 新零售凡是「想让用户掏钱」的生意都用得上。
Core Definition
三要素动态框架人（谁买）· 货（卖什么）· 场（在哪买），且核心角会转移。
Thinking Layer
商业型看生意的底层结构，先对齐三角再谈增长。

误区 vs 模型视角

Bad vs Good
✕

常见误区

Three Static Lists
  • 把人当「流量」「转化率」冷冰冰的数据标签
  • 货 = 摆上架标好价的物理商品，完事
  • 场 = 固定店铺 / 电商平台，是个「地方」
  • 三条腿各管各的，从没问过「对齐了吗」
例：东西不错就是没人买，有人逛就是不掏钱——三条腿没对齐，还怪流量贵。
✓

模型视角

A Rotating Triangle
  • 人 = 活生生的人，是货和场的「指挥官」
  • 货 = 产品＋体验＋服务＋社交货币，四位一体
  • 场 = 用户注意力里的任何场景，不在地图上
  • 三条腿对齐度第一，单条腿第一没用
例：山姆没在任何单条腿做到第一，但三条腿的匹配度做到极致，人对齐了增长自然来。

五个关键要点

The Five Insights
01

人：从消费者变成生活设计师

追求品质又要性价比、要个性又怕选错、重体验又懒得做功课——矛盾综合体。他买的不是你的产品，是他想象中更好的自己。

02

货：从商品变成解决方案

现在的货是「产品＋体验＋服务＋社交货币」四位一体。如果货还只是物理商品，你已经输了一半。

想清楚：这块积木，在用户的人生大厦里放哪一层？
03

场：从渠道变成生活场景

直播间、微信群、朋友圈广告、快闪店、甚至一篇小红书笔记，都是场。场不在地图上，在用户的注意力里。

04

核心角一直在转移

传统零售「场」为核心（地段），电商时代「货」为核心（SKU＋物流），数字时代「人」为核心（以用户倒推货和场）。

05

三角凳隐喻

缺一条腿就塌。大多数商家的问题不是都不行，是一条腿特别短——补最短那条，比把长腿拉更长有用。

三步行动清单

Put It to Work

画你自己的三角

写下你的「人」是谁、「货」是什么、「场」在哪。诚实回答——对齐了吗？

补最短那条腿

大多数商家不是都不行，是一条腿特别短。人对但货不行？货好但场没找对？先补最短的。

用「用户的一天」重审场

别想「我的店 / 号 / 平台」，改想目标用户从睁眼到睡觉，哪些瞬间有购买冲动？那些瞬间就是你真正的场。

模型金句

In a Nutshell

人货场的本质不是三件事，是一个问题——你到底围绕谁在转。

岛主批注

场不在地图上，在用户的注意力里。谁抢到时间和情绪，谁就拥有场。

场的重新定义

对齐大于单点第一

马上能用的事

Tomorrow's Moves
✓

重画你的三角

标出人 / 货 / 场，并诚实圈出最强和最弱的那条腿。

✓

把「用户画像」换成「用户的一天」

别只贴标签，去理解他怎么活、怎么消费，人是活的。

✓

检视你的货

它解决了什么问题、带来什么感觉、能给什么社交货币？只卖砖头就输一半。

✓

列出用户一天的冲动瞬间

那些瞬间就是你的真场，别只盯着店铺和 App。

✓

别追单条腿第一

先让三条腿对齐，匹配度比单点极致更决定生死。
