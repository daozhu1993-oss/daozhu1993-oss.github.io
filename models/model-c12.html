一句话总览： 别跟巨头抢山头，去他看不上的缝隙里生长。不和巨头抢同一个山头，而是在他忽视的缝隙里长出自己的第二曲线。巨头像一头大象，踩不死蚂蚁，是因为蚂蚁在它看不见的脚趾缝里，那地方对它来说太小，对你来说刚好安家。
Origin
李善友《第二曲线创新》把"错位竞争"讲透：避开主战场，在缝隙长第二曲线。
Field
战略 / 竞争策略小公司对抗巨头、资源有限时的破局法。
Core Definition
五步错位法确认赛道 → 了解巨头 → 寻找机会 → 组合产品 → 尝试测验。
Thinking Layer
战略定位层 / 竞争策略在"怎么赢"之前，先选"在哪赢"。

正面对刚 vs 错位生长

Bad vs Good
✕

正面对刚

Head-on, Doomed
  • 跟巨头硬刚同一赛道，越卷越没利润
  • 资源不够，只能当陪跑
  • 在同一赛道里卷到心酸
  • 比力气必输，还怪自己不够拼
例：拿着渔网去爬山，装备再好也使不上劲，白费力气。
✓

错位生长

Flank & Grow
  • 去巨头看不上的缝隙安家
  • 分析巨头各环节留了什么麻烦
  • 用红利 / 优势 / 错位三维度组合产品
  • 先做 MVP 验证，别一上来就 all in
例：拼多多避开淘宝京东主战场，五环外社交拼团三年八亿用户；瑞幸错位星巴克"第三空间"，打"快取+高性价比"，三年上万家店。

五步拆解

The Five Steps
01

确认赛道

依据你的爱好、特长、资源选行业。选错赛道，努力全是负分，别拿着渔网去爬山。

02

了解巨头

把在位企业从各环节拆开分析，看它为用户做了什么、又留下了什么麻烦。它的盲区，就是你的机会窗口。

03

寻找机会

真机会 = 市场大 + 利润高 + 可持续增长。伪机会是热闹完只剩一地鸡毛的"红海细分"。

04

组合产品

用红利、优势、错位三个维度，确认一个能满足需求的新产品。别贪全，先在一个尖上扎透。

05

尝试测验

造一个 MVP（最小可行产品），测试、改进、再测试，从 0 到 1，从 1 到 100。MVP 是低成本试错的小船，翻了也不沉底。

The Five-Step Flow
