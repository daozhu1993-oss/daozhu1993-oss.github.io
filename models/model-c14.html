品牌绝对价值象限

Brand Value Quadrant
岛主解读 / 体系  ·  结合 POM 模型  ·  品牌战略 / 价值定位
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 你的品牌溢价，到底靠哪条腿站着。品牌绝对价值象限把品牌价值拆成"人设强 / 弱 × 技术 / 创新强 / 弱"四个象限，而且必须结合 POM 模型一起看，才知道你驱动的是 P、O 还是 M。很多老板以为砸 O（营销）就能立品牌，结果钱花了，M（口碑）没起来，溢价始终悬空。
Origin
岛主解读 / 体系结合 POM 模型，把品牌价值拆成四象限。
Field
品牌战略 / 营销溢价设计、营销预算分配、品牌起步都用得上。
Core Definition
四象限拆品牌人设强 / 弱 × 技术 / 创新强 / 弱，且必看 POM 驱动。
Thinking Layer
品牌战略层 / 价值定位在"怎么卖"之前，先定"靠哪条腿站着"。

只砸 O vs 定象限驱动

Bad vs Good
✕

只砸 O

Spend on O, No Premium
  • 以为砸 O（营销）就能立品牌
  • 钱花了，M（口碑）没起来
  • 溢价始终悬空，像个没充气的气球
  • 用错力，钱花在刀背上
例：老板砸 O 立品牌，M 空白，用户只肯比价不肯掏溢价。
✓

定象限驱动

Pick Your Quadrant
  • 在象限图圈出现在位置
  • 看清能驱动的是 P / O / M，别用错力
  • 选定一条腿主攻，别四个象限都想占
  • 想走品牌驱动，先问有没有几十年耐心
例：瑞幸用"饱和式开店+补贴"走市场驱动；拼多多剑走偏锋做下沉社群，把 M×P 玩成护城河，三年行业第二。

四个象限

The Four Quadrants
Q1

品牌驱动（人设强 + 创新强）

爱马仕、茅台、宝马、徕卡。靠时间积累的人设，O（营销）最难驱动，得靠 M（口碑）慢慢养，像老酒越陈越香，急不得。

Q2

产品驱动（创新强 + 人设弱）

苹果、特斯拉、戴森、小米。硬科技就是最好的广告，产品自己会说话，P（个人体验）直接拉满，用户上手就懂好在哪。

Q3

渠道驱动（M×M）

P&G、伊利、永辉。铺货密度就是护城河，用户走到哪都能遇见你，靠"无处不在"的安全感，漏掉一个货架都算失守。

Q4

市场驱动（M×P）

瑞幸、盒马、拼多多。要么饱和式打击拼财力，要么剑走偏锋做下沉社群，靠 M（传播）和 P（低价感知）打穿，路径最野最猛。

提醒

多数人想走品牌驱动却没耐心

先想清自己落在哪个象限，别用错力，否则钱花在刀背上。
