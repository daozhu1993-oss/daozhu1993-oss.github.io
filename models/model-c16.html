马斯洛需求理论

Maslow's Hierarchy

用户买的从不是功能，是他那一层没被填满的洞

亚伯拉罕·马斯洛  ·  Abraham Maslow  ·  《动机与个性》(1970版)
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 马斯洛在《动机与个性》（1970版）里把人的动机拆成 7 层——人的行为是被需求推着走的，营销得对准那一层。需求像梯子，一层没踩稳，上一层就够不着，你往顶层喊，底层的人听不见。
Origin
亚伯拉罕·马斯洛Abraham Maslow，人本主义心理学奠基人，《动机与个性》(1970版)。
Field
动机心理学 / 营销 / 产品设计理解"人到底图什么"的底层框架。
Core Definition
需求分层，由低到高逐级满足生理 → 安全 → 归属 → 尊重 → 认知 → 审美 → 自我实现。
Thinking Layer
动机洞察层在"卖功能"之前，先看清用户卡在哪一层。

只卖功能 vs 对准需求层

Bad vs Good
✕

只卖功能

Feature Pitching
  • 参数拉满，却卖不动
  • 死磕价格战，利润薄得像纸
  • 没搞清人在这个阶段到底图什么
  • 底层没填满就喊顶层，用户听不见
例：用户连饭都愁时，你跟他讲审美，再好看也塌，他扭头去了别家。
✓

对准需求层

Target the Right Layer
  • 先画出用户卡在哪一需求层
  • 看竞品在哪层卷，找上层空白
  • 给产品加"意义"，而不只是参数
  • 精准扎进某一层，用户自然买单
例：奢侈品卖自我实现与尊重，小米卖性价比命中安全与归属，都精准扎进某一层，所以都赚到。

四组关键需求层

The Seven Layers, Grouped
01

生理 / 安全（基本需求）

吃喝拉撒、健康财产。底层不牢，上层免谈，就像房子没地基先谈装修，再好看也塌，用户连饭都愁时不会为审美买单。

02

归属与爱 / 尊重

家庭情感、自尊成就。人终究怕孤独、要被看见，这是社交产品的命门，也是"晒"这件事的源头——晒不是炫耀，是确认自己被接纳。

03

认知 / 审美（成长需求）

探索好奇、美感规律。吃饱了就想"好看又有意思"，这是内容消费和设计的引擎，用户开始为"舒服""高级"多付钱。

04

自我实现 / 超越

价值实现、天人合一。顶层是"我想成为谁"，奢侈品卖的就是这一口身份感，让你在镜子前觉得"这就是我"。

应用流程

From Feature to Meaning

定位需求层

画出你用户当前卡在哪一需求层，别一上来就堆功能。

找上层空白

看竞品都在哪层卷，找没被填满的那一层，那里利润往往更厚。

注入意义

给产品加一句对准该层的"意义"，而不是念一串参数。

定期复查

用户需求层会随阶段上移，定期回看产品还对不对准那一层。

模型金句

In a Nutshell

人不为功能买单，为"我是谁"买单。

马斯洛需求的核心

需求像梯子，一层没踩稳，上一层就够不着，你往顶层喊，底层的人听不见。

为什么别只卷价格

多数产品只盯着生理层打价格战，却没人去填尊重和自我实现那层，利润自然薄得像纸，还怪行业卷。
