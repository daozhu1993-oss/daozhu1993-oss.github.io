Ohmae's 3C Model

好战略得同时盯住三个C，缺一个都悬

大前研一  ·  Kenichi Ohmae  ·  日本战略研究领军人物
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 日本战略研究领军人物大前研一给出 3C 模型——好战略必须同时盯住三个 C：公司自身（Corporation）、顾客（Customer）、竞争者（Competition），缺一个都不稳。三个 C 像三脚架，少一脚相机就倒，照片再美也糊。
Origin
大前研一Kenichi Ohmae，日本战略研究领军人物，3C 模型提出者。
Field
战略 / 竞争分析制定"可持续优势"的底层框架。
Core Definition
战略 = 公司 × 顾客 × 竞争者三者交集里，藏着对手没占的空位。
Thinking Layer
战略整合层在单点思考之上，把三方放进同一张图。

只盯对手 vs 三C对齐

Bad vs Good
✕

只盯对手

Rival Myopia
  • 只盯着对手打，忘了顾客和自己
  • 跟风抄袭，结果被晾在一边
  • 本末倒置，顾客早跑了还不知道
  • 赢了参数，输了人心
例：新手最爱死盯竞品，结果顾客变了还不知道，赢了对手输了市场——这种"胜利的失败"最常见。
✓

三C对齐

Corporation · Customer · Competition
  • 同时看公司、顾客、竞争者
  • 找三者交集里对手没占的空位
  • 战略起点是顾客，不是对手
  • 缺一个 C，都是盲人摸象
例：优衣库用"平价优质"在 ZARA 和奢侈品牌之间错位，算清自己的供应链和用户要基础款，硬是长出护城河。

三个 C 与整合

The Three C's
C1

Company（自身）

你的资源和资金利用。先看清自己几斤几两，别拿短板去拼人家的长板，否则白送，像拿菜刀跟人比剑，输在起跑线。

C2

Customer（顾客）

用户的真实需求。战略的起点是顾客，不是对手——很多人本末倒置，盯着竞品改，顾客早跑了还不知道。

C3

Competition（竞争者）

对手手里的牌。你的差异化，恰恰来自对手的空白，他懒得做的，可能正是一群人的刚需，空白就是你的金矿。

+

整合三者

只有把公司、顾客、竞争者放进同一个战略里，可持续优势才存在。单看一个 C，都是盲人摸象，容易一叶障目。

应用流程

Align the Three C's

写三个 C

分别写下三个 C 的当前答案，别只凭印象。

找交集空位

找三者交集里对手没占的空位，那里是你的金矿。

对齐检验

战略每动一步，回看三个 C 是否还对齐，别跑偏。

别只盯竞品

定期确认顾客需求有没有变，赢了参数别输掉顾客。

模型金句

In a Nutshell

盯着对手的人，最容易丢掉顾客。
