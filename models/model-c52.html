Thinking Model · No.05

情景领导力

Situational Leadership
好领导不是一种样子，是照着人来变样子
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 没有最好的领导风格，只有最配下属"成熟度"的那一种。先判断对方在 D1–D4 哪一格，再把领导手法从手把手教调到放手授权——风格跟着人走，不跟着你舒服走。
Origin
保罗·赫塞 Paul Hersey20 世纪 60 年代提出，核心：风格匹配下属成熟度。
Field
领导力 / 团队管理 / 情境管理凡是要"照着人来变样子"的带人现场。
Core Definition
D1–D4 成熟度 × S1–S4 风格风格匹配成熟度，而非固定一套。
Thinking Layer
因人施策层 / 关系管理在"把事想清楚"之后，解决"怎么对人"。

一套打法 vs 照着人变

Bad vs Good
✕

同一套管理法

One Style Fits None
  • 对刚入职小孩事无巨细盯到他烦
  • 对老油条还像带新人一样手把手
  • 风格跟着自己舒服调，不跟着人调
  • 对熟手还盯着，等于羞辱他的专业
例：对资深设计天天改他稿（S1），他更摆烂。
✓

照着人来变样子

Style Follows Maturity
  • D1 高指导低支持（S1）手把手教
  • D2 高指导高支持（S2）既教又哄
  • D3 高支持低指导（S3）多听多鼓励
  • D4 低指导低支持（S4）授权放手
例：资深设计换 S4 全权丢他，只问"需要我挡什么"，准时交了。

四个成熟度，四种手法

D1–D4 & S1–S4
D1

没能力有意愿

像刚学骑车的热情娃。用 S1 高指导低支持，手把手教。这时候放手 = 害他。

D2

没能力没意愿

撞了墙泄气了。用 S2 高指导高支持，既教又哄。光骂不教，他直接躺。

D3

有能力没意愿

会干但不想干。用 S3 高支持低指导，多听多鼓励，把油门踩回来。别再教他怎么干，问他愿不愿意干。

D4

有能力有意愿

熟手。用 S4 低指导低支持，授权放手。还盯着他，是在羞辱他的专业。

应用流程

From Guess to Match

风格跟着成熟度调，不是跟着你舒服调，省力的那套不一定对。

人成长，手法退

人成长了，你的手法得跟着退，从 S1 一路退到 S4。

定期重评
