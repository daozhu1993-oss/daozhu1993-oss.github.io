组合创新进阶模型

Combinatorial Innovation
创新不是发明，是把旧零件重新拼  ·  约瑟夫·熊彼特  ·  "创新理论之父"
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 创新不是从零发明，是把已有要素重新拼一遍。根子上是熊彼特说的——把旧零件（渠道、方法、人群）往"十倍变化"的点上卡，错位组合，比从白纸硬想强得多。
Origin
约瑟夫·熊彼特Joseph Schumpeter，"创新理论之父"；岛主进阶为可落地的四步法。
Field
创新方法 / 产品运营产品创新、内容创作、商业模式、错位竞争都用得上。
Core Definition
旧要素的新组合拆成零件 → 找十倍变化 → 卡对单一要素。
Thinking Layer
创新方法层在"想创新"之后，解决"到底怎么动手拼"。

天才专利 vs 旧零件新拼

Genius vs Recombine
✕

把创新当天才专利

Blank Page Panic
  • 觉得创新是天才的专利，自己没那根筋
  • 老板一说"做个创新"，对着白纸发呆两小时
  • 追求从零发明、全新东西
  • 堆最多零件，反而拼不出花
例：硬想一个"从来没人做过"的方案，越想越空，最后一页没写出来。
✓

旧零件新拼

Recombine the Old
  • 先拆结构，把问题拆成一筐零件
  • 找十倍好 / 十倍速变化的现成要素
  • 匹配单一要素，做错位组合
  • 像搭乐高，卡对那一颗就够
例：费曼学习法 + 小红书短文拼一起，没发明新东西，反而比纯干货涨粉快三倍。

四个关键步骤

The Four Steps
01

定义问题，定目标

分两种：目标导向型（带着答案找路）和随机探索型（逛着逛着撞见）。岛主多走导向型：先定今天要讲清哪个模型。

02

拆解结构

用 PEST 看大环境，用波特五力看行业，把问题拆成一筐零件。拆得越细，能拼的花样越多。

03

找十倍好 / 十倍速变化

哪里有要素在十倍变好或十倍变快？新技术、新渠道都是现成零件。

04

匹配单一要素组合

对内看核心能力，对外做错位竞争。像搭乐高，不是堆最多，是卡对那一颗。

应用流程

Start Combining

定问题类型

写下要解决的问题，标明是导向型还是探索型。别上来就空想。

列三个旧零件

列出手边能用的三个"旧零件"：渠道、方法、人群都算。

找十倍变化点

找一个十倍变好或变快的点，把零件往那卡。

卡对那一颗

别追求全而新，卡对单一要素做错位组合就够。
