Thinking Model · No.05

组织心智

Organizational Mindset
李善友  ·  《第二曲线》  ·  组织认知与变革
一句话总览： 企业不是冷冰冰的机器，它也有自己的"脑子"，而且一样会固执、会盲。深埋在里面的"我们就是这样"的假设，像一副有色眼镜，挡住了所有和它冲突的提醒。
Origin
李善友《第二曲线》在《第二曲线》中重点阐述"组织心智"这一概念。
Field
组织行为 / 认知 / 变革凡要推动组织改变、打破惯性都用得上。
Core Definition
心智模式 = 有色眼镜组织也有乔哈里窗：公开 / 隐私 / 盲点 / 潜能。
Thinking Layer
认知层 / 组织觉察在"改什么"之前，先看见"为什么改不动"。

死守假设 vs 松动眼镜

Bad vs Good
✕

被心智绑死

Blinded by Mindset
  • 数据都红了，还坚称"我们一直这样没错"
  • 把预警挡在"盲点区"门外
  • 成功越久，眼镜越厚、越难摘
  • 不是没人提醒，是提醒进不来
例：一家厂死守"线下渠道为王"，电商来了硬说不靠谱，等反应过来份额丢了一半。
✓

松动心智眼镜

Loosen the Lens
  • 先承认组织也有"看不见的盲点区"
  • 主动找外部视角来戳锚点
  • 把"我们一直对"改成"先假设，再验证"
  • 让不同意见有进门的缝
例：定期请外部顾问挑战"我们一向…"的口头禅，盲点区才第一次被照到。

四个关键要点

The Four Disciplines
01

组织也有乔哈里窗

公开区、隐私区、盲点区、潜能区——组织和个人一样，有自己看不穿的那块。

判断：看一个组织卡在哪，就看它肯不肯碰自己的"盲点区"。
02

盲点区最危险

别人早看穿了、自己死活不信——这格不破，再好的建议也进不来。

例：市场都变了，管理层还信"便宜万能"，痛失拐点。
03

心智模式 = 有色眼镜

当年靠低价打下的江山，眼镜就写着"便宜万能"；当年靠线下，眼镜就写着"渠道为王"。

例：眼镜本是成功的原因，市场一变就成了障眼法。
04

"我们一向…"是锚点

组织里那些脱口而出的口头禅，往往就是心智锚点，也是松动的突破口。

例：把"我们一向重质量"列出来，再问：它现在还成立吗？

应用流程

Loosen the Lens

列出口头禅

把公司"我们一向…"的话写下来，那往往就是心智锚点。

引入外部视角

找几个局外人，专门来戳这些锚点的盲点区。

改掉"一直对"

把"我们一直对"改成"我们先假设，再验证"，给不同意见留门。

季度复盘锚点

每季回看这些口头禅，哪些还成立、哪些已成枷锁。

模型金句

In a Nutshell

个人会被经验绑住，组织会被成功绑死。

组织心智的残酷真相

不是没人提醒，是组织心智把提醒挡在了盲点区外。

为什么好建议进不来

当年让你赢的眼镜，市场一变，就成了让你看不见的障眼法。

岛主批注

马上能用的事

Put It to Work
✓

开一场"口头禅大会"

让团队把"我们一向…"全写下来，先看见锚点。

✓

请个局外人来挑刺

外部视角最擅长戳盲点区，自己人不好意思说。

✓

把"我们对"改"先验证"

一句话的语态转换，给不同意见留了门缝。

✓

给反对意见发"通行证"

谁提了被眼镜挡住的风险，奖励，而不是边缘化。

✓

季度复盘心智锚点

别等出事才摘眼镜，定期自己照一照。

✓

警惕"我们一直都行"

越成功时的口头禅，越可能是下一个坑的预告。

什么时候该掏出它

When to Use
组织变革 战略转型 文化重塑 跨部门协作卡点 老业务遇新对手 复盘"为什么没听劝" 任何"明明看见坑却绕不过去"的时刻
