Thinking Model · No.04

找到热爱一生的事业

Career Passion · Ikigai (Life Value Venn)
生命价值韦恩模型  ·  Ikigai  ·  生涯规划 · 自我探索
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 完美的天命事业，藏在四个圈的交汇处——你热爱的、你擅长的、世界需要的、有人愿意付费的。四圈重叠，就是传说中的 Ikigai（生命价值）：充实、有意义、还有回报。
Origin
日本「生き甲斐」思想Ikigai（生命价值），经生涯规划与积极心理学发扬，对应舒伯终生生涯发展思路。
Field
生涯规划 / 职业选择兴趣、能力、价值三方匹配，破解"工作还是天职"之问。
Core Definition
四圈交集 Ikigai热爱 Passion · 擅长 Profession · 世界需要 Mission · 有人付费 Vocation。
Thinking Layer
定位与选择层在"怎么做好"之前，先想清楚"做什么值得做"。

单圈挣扎 vs 四圈交汇

Bad vs Good
✕

缺角的状态

The Lopsided Zones
  • 只热爱+擅长→"自嗨"，缺社会价值与回报
  • 只热爱+世界需要→"美好理想"，不擅长难持续
  • 只擅长+有人付费→"空虚赚钱机器"，职业倦怠
  • 只世界需要+付费→"一份苦差"，为生存所迫
例：擅长枯燥的数据分析，收入高却毫无激情——典型的"专业"区，空虚。
✓

Ikigai 天命

The Sweet Spot
  • 四圈重叠：热爱且擅长，世界需要且有人付费
  • 不再"工作"，而是在实践"天职"
  • 充实、满足、有意义，还有回报
  • 是动态指南针，不必一步到位
例：你做热爱且擅长的事，它恰好是市场要的、客户愿买单——这就是"命中心"。

四个核心圈

The Four Circles
P

你热爱什么 Passion

做什么会忘记时间、充满喜悦？热情的来源。缺了它，事业只是"苦差"或"赚钱机器"。

例：热爱的+擅长的=「热情」，但没人买单就可能陷入自嗨。
P

你擅长什么 Profession

做什么常被夸奖认可？能力的来源。缺了它，理想再美也难持续落地。

例：擅长的+有人付费=「专业」，稳定收入但可能毫无激情。
M

世界需要什么 Mission

身边或世界最需要解决什么问题？意义的来源。让你的技能连接更大的价值。

例：热爱的+世界需要=「使命」，充满意义感却可能缺能力变现。
V

有人付费吗 Vocation

哪些产出已被证明有人愿买单？市场的来源。把价值产品化、服务化。

例：世界需要+有人付费=「工作」，能温饱却可能不喜欢不擅长。

三步找到你的 Ikigai

Find It in 3 Steps

第一步：诚实回答四问

拿纸笔写下：做什么会忘时间？被夸什么？世界最需解决什么？哪些技能已被证明有人付费？

第二步：找交集与缺口

对照答案，你最接近哪个区域？阻碍你到中心的，主要缺了哪一环——热爱、变现还是价值？

第三步：微小行动计划

缺热爱就每周投 2 小时去试；缺擅长就针对性练；缺意义就连社会；缺付费就研究市场、做产品化。

模型金句

In a Nutshell

Ikigai 不是固定的终点，而是一个动态的指南针——随着阅历，四个圈的答案都会变。

模型的本质

不必强求立刻坐在最中心，但可以明确朝着中心的方向，挪动你的每一步。

行动心法

从一份"工作"，升级为一份"专业"，再注入"热情"和"使命"，最终无限接近属于你的生命价值。

岛主批注

马上能用的事

Put It to Work
✓

画一张四圈图

纸上画四个相交圆，分别写下热爱/擅长/需要/付费，重叠区一眼可见。

✓

先做"自我诊断"

你现在处在热情、使命、专业、工作哪一区？承认位置是挪动的前提。

✓

缺哪环补哪环

缺热爱去体验，缺擅长去练，缺意义去连接社会，缺付费去研究市场。

✓

每周投 2 小时给热爱

不必裸辞，先让"热情"那圈长大，中心会慢慢靠近。

✓

把能力产品化

你被夸的技能，能否变成别人愿付费的东西？这是离"付费圈"最近的一步。

✓

接受它是动态的

别焦虑"还没找到"，指南针本就在转——方向对，就值了。
