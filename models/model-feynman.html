费曼学习法

Feynman Technique
理查德·费曼  ·  Richard Feynman  ·  物理学家自学绝招 · 输出倒逼输入
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 费曼学习法被公认为"世界上最高效的学习法"，核心简单到惊人：如果你不能用简单的话把一个概念讲给外行听，说明你根本没懂。它的本质是——输出倒逼输入，教别人就是最好的学。
Origin
理查德·费曼Richard Feynman，诺贝尔物理学奖得主的自学绝招。
Field
学习方法 / 知识管理凡是要"真学会"而不是"假看过"的场合都用得上。
Core Definition
以教促学讲不明白 = 没真懂，用"教"的标准要求"学"。
Thinking Layer
学习 / 认知层在"输入"之后，解决"怎么才算真的懂"。

假懂 vs 费曼

Bad vs Good
✕

用输入欺骗自己

Input Illusion
  • 读完一本书，合上只记得书名
  • 听完课觉得"太有道理"，一问就卡壳
  • 只收藏不输出，用"学了"自我安慰
  • 被动记笔记，从没检验过真懂没
例：读了一堆干货，别人问起支支吾吾，真要用时脑子一片空白。
✓

费曼学习法

Output Forces Input
  • 想象在给完全不懂的人上课
  • 讲不清楚的地方，就是没真懂的盲区
  • 回去重学，再简化，直到能讲顺
  • 用生活化例子，把知识内化到身上
例：看完岛主笔记，用自己的话在评论区解释一遍——讲得出来，才是真掌握。

五个关键要点

The Five Points
01

确立目标

越具体越好。不是"我想学经济学"，而是"我要搞懂'机会成本'这个概念"。目标清晰，才能聚焦。

02

深度分析

搜集资料、阅读、思考，直到"我好像懂了"。注意：这是传统学习的终点，却是费曼法的起点。

03

组织

把内容重新排列、总结、归纳，准备你的"教案"：怎么引入、怎么展开、怎么举例。

04

简化

假装教一个完全不懂的人（8 岁小孩或隔壁大妈）。能用大白话讲清，才算过；卡住就回去重学。

标准：生活化例子 + 简单语言，把这个概念讲清楚。
05

内化

反复这个过程，直到概念成为你思维的一部分，任何场景下都能随时调用，不用翻书。

★

核心判据

讲不明白，就不算懂。费曼的精髓不是"教别人"，而是用"教"的标准来要求"学"。

五步循环：一张图学会费曼

The Feynman Loop

确立目标

先问自己：到底要学什么？对象越具体，后面越聚焦。

深度分析

读到"我好像懂了"——别停，这只是费曼法的起点。

组织

重排归纳，备好你的"教案"：逻辑顺序、引入、展开、举例。

简化

用最通俗的话讲给外行。卡住就回去重学，再简化，直到讲顺。

内化

反复到知识长在身上，随时能用自己话讲明白，不求人。

岛主批注

马上能用的事

Put It to Work
✓

读书后合书讲三句

读完一章，合上书用三句话把核心讲给自己听（可录音），讲不出就是没懂。

✓

学完模型评论区复述

看完笔记，用自己话在评论区解释一遍，就当教我，这是检验真懂的最快方式。

✓

学新方法讲给同事

他们听懂了、能用了，说明你真正掌握了，不是自以为懂。

✓

每次"我懂了"先自问

能讲给外行听吗？讲不明白，就老老实实回去再学。

✓

卡住处标记重学

讲不顺的地方，正是盲区，回去补完再简化，别跳过。

✓

用生活例子替换术语

把专业词翻成大白话和日常类比，能通俗讲清才是真内化。
