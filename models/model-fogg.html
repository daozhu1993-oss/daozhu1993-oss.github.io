福格行为模型

Fogg Behavior Model
BJ Fogg  ·  斯坦福行为科学家  ·  20 年研究 · B = MAP
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 福格行为模型用一个等式说清你为什么做不到想做的事：B = MAP，行为 = 动机 × 能力 × 提示。三要素缺一个，行为就不会发生。所以别再骂自己"没毅力"——你不是懒，你是设计错了。
Origin
BJ Fogg斯坦福大学教授，花 20 年研究行为如何真正发生。
Field
行为设计 / 习惯养成 / 心理学想改变习惯、立住 flag 的人必备。
Core Definition
B = MAP行为 = 动机(M) × 能力(A) × 提示(P)，任一为 0 结果即 0。
Thinking Layer
心理 / 行为层在"想改变"之后，解决"怎么让改变真发生"。

靠毅力 vs 福格设计

Bad vs Good
✕

只靠意志力

Willpower Alone
  • 把失败归咎"没毅力"，不断自我攻击
  • 一上来定大目标：每天跑 5 公里
  • 只打鸡血、看励志视频，靠动机行船
  • 完全忽视"提示"，没人提醒就忘
例：年初立 flag 办健身卡，动机满格；一周后加班累瘫，动机掉到谷底，计划崩了。
✓

福格行为设计

Design, Don't Demonize
  • 看懂 B=MAP，三要素缺一不可
  • 降门槛做"微习惯"，先跨过开始
  • 用"如果…那么…"把新习惯锚定旧习惯
  • 庆祝微成功，把新行为和愉悦绑定
例：每天做 1 个俯卧撑（能力够），刷完牙就做（提示在），做完夸自己"干得漂亮"。

六个关键要点

The Six Keys
M

动机：会波动

你想做这件事的渴望。问题不在它弱，在它像潮水有涨有落——只依赖动机，永远到不了彼岸。

A

能力：关键在"更容易"

福格最颠覆的一点：改变的关键不在"更想"，在"更容易"。把目标缩到不可能失败，先跨过"开始"。

例：别定"每天读 1 小时"，先定"每天读一页"。
P

提示：最被忽视

什么触发了行动？最强提示法是"如果…那么…"：把新行为锚定在一个你已会做的事上。

=

公式：任一为 0 即 0

行为能否发生 = 动机 × 容易度 × 有没有人提醒。三者相乘，任何一个归零，结果就是零。

微

微习惯

缩到"这有什么难的"程度，你就没有借口不做。一旦做了，最难的坎"开始"就被你跨过。

庆

庆祝微成功

做完立刻在心里说"干得漂亮"。这是训练大脑把新行为和愉悦绑定——微习惯能持续的秘密燃料。

落地四步：立刻设计自己

Design Yourself

缩到不可能失败

"每周运动 3 次" → "每天做 1 个俯卧撑"。小到没借口，先动起来。

找个锚点挂上去

用"如果…那么…"：如果刷完牙，那么做 1 个俯卧撑。旧习惯当新习惯的触发器。

庆祝每个微成功

做完立刻在心里夸自己"干得漂亮"。这不是矫情，是在给行为绑上愉悦燃料。

检查三要素

动机、能力、提示都齐了吗？差哪个补哪个，别再只怪自己没毅力。

模型金句

In a Nutshell

别再骂自己"没毅力"了。你不是懒，你是设计错了。

福格最狠的一句

改变的关键不在"更想"，在"更容易"。

能力的颠覆性

从今天起，别再问"我为什么做不到"，改问"我怎么把它设计得更简单"。

岛主批注

马上能用的事

Put It to Work
✓

把大目标缩成一个微动作

"每天跑 5 公里"改成"每天 1 个俯卧撑"，先让"开始"发生。

✓

给新习惯找个锚点

用"如果…那么…"绑到旧习惯上：刷完牙、喝完水就触发。

✓

做完立刻庆祝一次

心里说句"干得漂亮"，把愉悦感焊进这个新行为。

✓

把自责改成"哪里设计错了"

没做到别骂自己，去查动机/能力/提示哪个断链了。

✓

三要素齐了没？

想养成习惯前，先确认动机、门槛、提示三者都到位。

✓

旧习惯当触发器

已有的日常就是最好的提示源，别另设闹钟硬提醒。

什么时候该掏出它

When to Use
