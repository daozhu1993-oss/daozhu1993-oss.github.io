学习金字塔

Learning Pyramid

← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 学习金字塔把学习方式分成两大类七层级，对应不同的「平均留存率」——被动听讲只留 5%，主动教人能留 90%。它是戳破"假装学习"幻觉的照妖镜，也是帮你真正"学进去"的路线图。
Origin
埃德加·戴尔（Edgar Dale）"经验之锥"；美国国家训练实验室（NTL）的留存率研究。
Field
学习方法 / 认知提升 / 知识管理学生、职场人、任何"学了就忘"的人。
Core Definition
两类七层级留存率被动区（听讲5%→演示30%）+ 主动区（讨论50%→实践75%→教人90%）。
Thinking Layer
认知效率层在"学了"之后，专门解决"记住了多少、用得出来吗"。

假装学习 vs 真学习

Passive vs Active
✕

假装学习（被动区）

Pretend Learning
  • 坐在那听，以为自己懂了
  • 划重点、做标记，"熟悉感"骗自己
  • 看视频刷纪录片，大脑其实没动
  • 收藏 100 篇干货，从不打开第二遍
例：读了一本书，一周后只记得书名——留存率 5%–10%，白忙一场。
✓

真学习（主动区）

Real Learning
  • 讨论辩论，被迫组织语言、发现漏洞
  • 动手做、去犯错、去复盘，身体记忆更持久
  • 教人/马上用，逼自己真正弄懂
  • 用输出倒逼输入，留存率最高 90%
例：学完一个模型，立刻找生活场景用一次、讲给朋友听——留存率跳到 75%–90%。

七个层级 · 留存率

Seven Tiers of Retention
5%

听讲

最舒适也最易忘。坐在那听老师/博主讲，以为懂了——其实两周后所剩无几。

10%

阅读

划重点、做标记、反复看。你以为记住了，其实只是"熟悉感"在骗你。

20%

视听

看视频、听音频、刷纪录片。声光电刺激让你"感觉学到了"，大脑却没动。

30%

演示

看别人示范一遍，觉得"原来如此简单"。旁观者的"我懂了"，往往是最大的错觉。

50%

讨论

和别人交流、辩论、碰撞观点。你被迫组织语言、捍卫观点、发现自己的漏洞。

75–90%

实践 · 教人

动手做、去犯错、去复盘（75%）；把学到的讲给别人听或立刻用（90%）。为讲清，你必须真懂；为用出，你必须内化。

铁律：被动像用漏勺舀水，主动像用桶装水。

把新知识逼进主动区

Push Into the Active Zone

读完写一句理解

任何文章读完，写一句自己的话——这是"教人"前的准备，逼大脑开工。

学完用一次

学完一个模型，找一个生活场景实打实用一次（实践，75%）。

听完后讲一遍

听完一门课，找一个人讲一遍（教人，90%）——讲不清的地方就是你没懂的。

用输出倒逼输入

承诺把它写成分享或笔记。输出是最好的输入，逼你爬到金字塔顶端。

模型金句

In a Nutshell

被动学习像用漏勺舀水，主动学习像用桶装水。

两种学习方式的写照

学习的目的是"改变"，而不是"知道"。

学习的终点

输出是最好的输入——这正是为什么"教人"的留存率高达 90%。

岛主批注

马上能用的事

Put It to Work
✓

警惕"收藏即学到"

收藏、点赞、标记只是起点，问自己一句：然后呢？

✓

读完写句自己的理解

哪怕一句话，也算把内容拖进了主动区。

✓

学完找场景用一次

知行合一，身体的记忆比大脑更持久。

✓

听完课讲给一个人

讲不通，就是你还没真懂——立刻回去补。

✓

承诺一次输出

写笔记、做分享、发帖子，用输出锁住输入。

✓

把被动内容改造成主动

看视频时顺手记"我能用在哪"，别只当背景音。

什么时候该掏出它

When to Use
高效学习 学生备考 职场充电 读书内化 知识管理 内容创作 任何"学了就忘"的人
