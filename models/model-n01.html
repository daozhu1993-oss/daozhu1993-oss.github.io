群体领袖说服模型

Group Leader Persuasion Model
领袖说服群众靠的不是道理，是断言、重复、传染三板斧，外加五把刀。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 群体一旦聚到一起，智力就自动归零，只跟着情绪和暗示走。领袖要人信，靠的是断言、重复、传染，而不是逻辑。
Origin
勒庞《乌合之众》法国社会心理学家对群体心理的经典拆解
Field
群体心理 · 说服传播传销、谣言、crowd 动员都走这套
Core Definition
领袖用断言、重复、传染三板斧加权威/情绪/利益等五把刀，让群体情绪化盲从。人们要的是确定性，不是真相
Thinking Layer
认知防洗脑 · 批判性思维

误用 vs 善用

Bad vs Good
✕

把人群当脑子使

Treat the Crowd as Rational
  • 迷信绝对化口号，听到“包治百病”就热血上头
  • 只刷一个圈子的信息，被重复洗成了信仰
  • 把对方画的饼当现实，忘了算账
  • 以为自己想通了，其实是被传染
例：权健老总一句“没有我调理不好的癌症”，正中绝症患者求确定性的死穴，多少人把棺材本都奉上。
✓

把人群当情绪看

Read the Crowd as Emotion
  • 听到绝对化口号先愣三秒，追问“证据呢”
  • 主动看跟自己唱反调的信息源，破传染
  • 遇画饼先算账：落地要多少真金白银
  • 守住信息源多元，别被单一回路焊死
例：看到“稳赚不赔”的号召，先问谁在背书、要不要先交钱、退路在哪——而不是立刻上车。

领袖的三板斧与五把刀

The Axes and Knives
01

断言 · 第一板斧

给个斩钉截铁的结论，不解释、不举证。“包治百病”四个字，比医生三页诊断书都有杀伤力。人想要的从来不是真相，是确定性。

02

重复 · 第二板斧

谎言重复千遍成了真理，是在给神经“焊”回路，焊到最后你以为是自己想出来的。听得多了，假的也成了信仰。

03

传染 · 第三板斧

断言被人信了、重复被人传了，流行意见就生了根，像感冒一样一人传一人。群体里没有思考，只有传染。

04

五把刀

诉诸权威（找专家背书）、动之以情（激发羞愧恐惧）、晓之以理（把道理压成“语言钉”）、诱之以利（画饼＋金钱）、藏锋于剑（制造恐惧再指明方向）。

例：真正的洗脑，是这五样一起上，单用一招聪明人还来得及反应。
05

组合才致命

三板斧负责“让人信”，五把刀负责“让人动”。分开看都不稀奇，叠在一起就焊死了一套服从回路。

三步不被带走

Don't Get Carried Away

识别信号

听到“绝对、肯定、百分百”这类词先愣三秒，问自己：证据呢？谁在用它让我信？

守住信息源

别只刷一个圈子的信息，主动看跟自己唱反调的信源，破掉单一重复回路。

画饼先算账

对方承诺的梦，落地需要多少真金白银？把“愿景”翻译成“成本”再点头。

留痕复盘

每次被带节奏后记下“是哪招戳中我”，下次就有抗体，不被同一把刀再割一次。

马上能用的事

Put It to Work
✓

广告先反问三条

下次刷到“稳赚、包治”这类广告，先截图反问：证据、退路、谁在受益，再决定。

✓

关注两个反方号

关注两个和你立场相反的账号，每周各看一次，故意打破信息同温层。

✓

讲一遍防洗脑

给家人或朋友讲一遍“断言＋重复＋传染”，当一针免费疫苗。

✓

画饼先写成本

遇到别人画的大饼，先写一张“落地成本清单”再点头答应。

✓

建个带节奏本

建个“我被带节奏”小本，记满三次，你基本就长记性了。
