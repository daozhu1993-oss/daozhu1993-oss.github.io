网络模型

Network Model
你不是输在产品，是输在没接入那张看不见的网。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 近二十年最被低估的发现是"网络"——决定成败的往往不是你多努力，而是你站在网络里的什么位置。你的连接数、连接质量，比个人能力更早决定你的上限。
Origin
斯科特·佩奇《模型思维》 复杂系统学者将"网络"列为最具影响力的现代发现之一；岛主解读 / 思维模型合集
Field
网络科学 / 社会科学 从人脉经营到平台战略、个人成长都能用
Core Definition
节点与连接构成的系统 节点重要性取决于"度"（连接数）与所处结构位置；枢纽 vs 叶子，位置决定价值
Thinking Layer
战略 / 认知 看清自己在系统里的坐标，再决定怎么动

误用 vs 善用

Bad vs Good
✕

只经营圈内好友

Only Strong Ties
  • 把人脉当成"点赞之交"，只维护几个密友，圈子越来越封闭
  • 以为努力就够了，忽略自己在网络里是"叶子"还是"枢纽"
  • 看到别人赢家通吃，归因为运气好或产品好，没看见网络效应
  • 在幂律网络里当陪跑，还以为是自己不够努力
例：辛辛苦苦做内容，只发在几个熟人微信群，三个月零涨粉，以为是自己写得差。
✓

攒弱连接，抢枢纽位

Build Weak Ties & Hubs
  • 主动跨行业认识人，把"朋友的朋友的朋友"变成你的信息桥
  • 做内容、做产品尽早入场，抢"枢纽"位置，先发者通吃
  • 用梅特卡夫定律算账：网络价值随节点数平方涨，10 倍规模≈100 倍价值
  • 看清自己处在哪类网络（随机 / 小世界 / 无标度），别用错玩法
例：岛主一条跨界群里的弱连接，一次就引来了几十个精准关注，比圈内好友转发管用得多。

关键要点

Key Points
01

度（Degree）：你是枢纽还是叶子？

度就是你的连接数，也就是朋友数。枢纽节点连着无数人，普通节点连着几个——你的度越高，越难被替代。

02

弱连接是出圈桥

密友之间往往也是朋友（聚类系数高），但真正帮你找工作的，常是"朋友的朋友的朋友"。三度人脉理论上能到 25 万，且分布在不同城市、不同行业。

03

无标度与富者越富

新节点优先连大节点（偏好连接），流量、资源向头部集中。短视频先发者占了坑就稳了，晚入场只能陪跑。

04

梅特卡夫定律

网络价值随节点数平方涨，10 倍规模约等于 100 倍价值。所以"第一名吃肉，第十名喝风"不是运气，是数学必然。

怎么用起来

How to Use

画清你的网络位置

盘点自己处在哪类网络，是枢纽还是叶子，连接质量如何、覆盖哪些行业。

主动攒弱连接

跨行业、跨城市认识人，给信息留出"出圈桥"，别只在熟人圈打转。

抢枢纽早入场

做内容、做产品尽早占位，借偏好连接吃先发红利，别等赛道满了才来。

别在幂律里当陪跑

认清网络规则，在幂律网络掉队可能是结构问题，别全怪自己不够努力。
