非线性模型

Nonlinear Model
世界是曲线的，用直线思维期待回报，注定被现实打脸。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 线性是"叠加"，非线性是"互相放大或抵消"。你的努力、投资、成长大多是非线性的——前期平缓、后期爆发，或前期猛、后期骤降。看错曲线形状，就预测错结果。
Origin
数学与系统科学 对"整体≠部分之和"的建模，复利、72 法则、索洛模型是其应用；岛主解读 / 思维模型合集
Field
系统思维 / 经济学 投资、学习、管理都绕不开
Core Definition
变量相互作用，结果被放大或反转 凸函数斜率递增、凹函数斜率递减；不是简单相加
Thinking Layer
认知 / 决策 校准你对"回报节奏"的预期

误用 vs 善用

Bad vs Good
✕

用直线期待一切

Linear Expectation
  • 以为"多干一小时就多赚一小时"，把工资逻辑套到创业、投资上
  • 项目前三个月没起色就放弃，看不懂指数曲线最骗人的开头
  • 暴跌时还用线性思维"抄底"，没给非线性留一点缓冲
  • 相信"延长工时=持续增长"，忽略体力和回报的半衰期
例：手机"还能用"就不换，殊不知它每年折旧一半，真实价值早悄悄腰斩。
✓

看清曲线再下注

Read the Curve
  • 先判断面前是线性还是非线性，别用错预期模型
  • 想指数增长就押注复利与创新，而非拼命堆叠时长
  • 用 72 法则算翻倍：年利 9% 约 8 年翻一倍，给时间以耐心
  • 给非线性留缓冲，暴跌不慌、爆发不飘，尊重曲线节奏
例：岛主见过太多人项目前三月毫无起色就放弃，殊不知指数曲线最骗人的就是开头那段平平无奇。

关键要点

Key Points
01

整体 ≠ 部分之和

线性是叠加，非线性是相互作用——结果会被放大、抵消或反转。存银行是线性，炒股是非线性。

02

凸函数与凹函数

斜率递增（人群里能结的"对"暴增）是凸，斜率递减（冰激凌吃第三盒没味）是凹。判断关键：这事是叠加，还是互相放大？

03

指数增长最骗开头

棋盘放米第 64 格是 18.4 亿吨；复利同理，前期慢得可笑、后期快得吓人。好项目早期没人信，正因如此。

04

半衰期与索洛模型

手机每年折旧一半，是半衰期。长期增长只靠创新，延长工时只会低效——个人持续增长靠学习力。

怎么用起来

How to Use

识别曲线类型

问自己：这事是叠加，还是互相放大 / 抵消？先把曲线形状认出来。

押注复利与创新

想指数增长，就投时间和学习力，而非堆时长、熬工时。

用 72 法则做耐心

年利 9% 约 8 年翻倍，给好项目足够长的 gestation 期，别三月就撤。

给非线性留缓冲

暴跌别抄底、爆发别膨胀，尊重曲线的非线性，留好安全垫。

In a Nutshell
