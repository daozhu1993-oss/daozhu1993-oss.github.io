信号模型

Signal Model
别人看不清你，只能看你发出的信号——所以亮真底色，别只顾包装。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 竞争里，你的"隐藏属性"（实力、财富、人品）得靠信号透出来。信号的价值不在东西本身，而在它的成本——强者发信号便宜、弱者发不起，于是机制就能自动把人分开。
Origin
凡勃仑与《模型思维》 凡勃仑的"炫耀性消费" + 信号传递理论；岛主解读 / 思维模型合集
Field
经济学 / 博弈论 求职、营销、社交识人都用得上
Core Definition
用高成本行为证明隐藏属性 成本差异自动筛选强者；连续信号 vs 离散信号
Thinking Layer
认知 / 战略 看懂别人怎么"亮底牌"

误用 vs 善用

Bad vs Good
✕

只做表面包装

Cheap Packaging
  • 以为买件贵衣服、租辆豪车就能证明实力
  • 分不清真信号和假信号，被 P2P 奢华办公室唬住
  • 自己发信号靠铺张，烧钱却没有"连续可验证"的底色
  • 以为"看起来专业"的设备能骗过真同行
例：岛主刚工作跟风买"看起来专业"的设备，后来才懂：那是给外行看的信号，真同行一眼看穿。
✓

发高成本真信号

Send Costly, Honest Signals
  • 想被识别，发"高成本且连续"的信号，而非一次性装样
  • 识人时辨真伪：租来的豪车、奢华办公室都是可拆穿的假信号
  • 自己走正道——团体运动证健康，公益证慷慨，不靠铺张
  • 看懂机制设计：奖学金设多少，就筛选出哪类人
例：学历就是连续信号——微积分不治病，但它证明你扛得住整套知识体系，这才是真底色。

关键要点

Key Points
01

炫耀性消费证身份

凡勃仑早看透：人靠衣服、包、车、酒推断别人的隐藏属性。开贵车→大概率有财富，博士学历→大概率聪明。

02

离散信号自动分人

强者发信号成本低（c）、弱者成本高（C）。奖学金设 100 万人人申（混同），设 20 万只有强者申（分离）。

03

连续信号烧得起才有效

贵表、超跑、稀世珠宝，穷人发不起、强者能连续发。雄孔雀尾羽没用还惹祸，但雌孔雀远远就能看见——演化里"被看见"压倒"有用"。

04

学历是知识体系信号

微积分不治玻但它证明你扛得住知识训练。看人别只看标签，要看标签背后的真本事。

怎么用起来

How to Use

想被识别发真信号

选"高成本且连续"的方式，别一次性装样，经不起时间检验。

识人先辨真伪

警惕租来的豪车、奢华办公室这类可拆穿的假信号，多核实。

设计你的机制

想筛选谁，就把信号成本设到只有目标人群发得起。

走正道发信号

用团体运动证健康、公益证慷慨，不靠铺张浪费去亮身份。
