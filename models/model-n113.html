广播、扩散和传染模型

Broadcast, Diffusion & Contagion Model
信息靠讲、病毒靠防——懂传播底层法则，既能点火也能灭火。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 传播有三条不同的路：广播是一对多"刷存在感"，扩散是多对多"靠值得分享"，传染是 SIR 模型里 R0 定生死。搞错路线，广告费打水漂；用对路线，用户替你跑腿。
Origin
传播学 / 流行病学 对"信息、病毒如何扩散"的建模（如 SIR 模型）；岛主解读 / 思维模型合集
Field
传播学 / 营销 推广、舆情、公共卫生防疫都适用
Core Definition
三类传播机制 广播（一对多）、扩散（多对多分享）、传染（SIR 易感-感染-康复）
Thinking Layer
战略 / 认知 选对传播路线，再谈执行

误用 vs 善用

Bad vs Good
✕

一条路走到黑

One Channel Only
  • 产品没人主动聊，还死磕"口碑裂变"，不肯打广告
  • 只堆功能不讲故事，扩散概率低到没人愿意转
  • 当谣言和负能量的超级传播者，没意识到自己 R0>1
  • 防疫只看一面，忘了 R0 决定传播生死
例：三线电视台的保健品靠口碑根本卖不动，却还幻想"用户会主动聊"，只能一遍遍广播。
✓

三路组合拳

Mix the Three Channels
  • 推产品两手抓：持续广播打底 + 发动用户扩散
  • 想扩散就造"新奇故事"，让内容自己长腿跑（陈胜鱼肚藏帛书是祖宗级玩法）
  • 用 SIR 思路防疫：降接触、减感染、提痊愈、护易感，盯住 R0
  • 警惕自己成为超级传播者，别扩散谣言和负能量
例：小米雷军"专注极致口碑快"七字诀，把好产品变成用户主动转介绍——扩散概率拉满。

关键要点

Key Points
01

广播：一对多刷存在感

触达比例叫广播概率，速度递减所以要持续投——用户不主动聊，就老老实实打广告。

02

扩散：多对多靠分享

扩散概率=分享概率×接触概率。故事越新奇，越像病毒自己长腿跑。

03

传染 SIR：R0 定生死

易感→感染→康复，R0>1 传遍、<1 消失。超级传播者（传染超 10 人）是关键节点。

04

新冠四招对应模型

降接触、减感染、提痊愈、护易感——正是把 SIR 拆成具体动作。

怎么用起来

How to Use

判断你的东西会不会被聊

用户不主动聊，就先靠广播打底，别硬等口碑。

造新奇故事

想扩散就编离奇、有梗的内容，拉高分享概率。

算清 R0 盯节点

分清易感 / 感染 / 康复，抓住超级传播者重点经营。

别当负能量源头

管住自己，不扩散谣言和情绪垃圾，避免 R0>1。
