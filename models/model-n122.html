均值回归

Regression to the Mean
极端表现终会向价值中枢靠拢，巅峰与谷底都只是路过，别把回归误当成努力或失误。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 均值回归告诉我们，极端值（运气、业绩、行情）终会向长期中枢回落——别把回归误当成自己努力或失误的功劳，也别在高峰低估、谷底绝望。
Origin
弗朗西斯·高尔顿 · 统计 / 遗传学源自 1078 对父子身高的豌豆与身高研究
Field
统计学 / 投资 / 认知金融、教育、绩效评估
Core Definition
极端值终会向"价值中枢"靠拢的规律，如引力般把偏离拉回运气不遗传，极端不可持续
Thinking Layer
认知反直觉、概率思维

误用 vs 善用

Bad vs Good
✕

错误归因

Misattribution
  • 孩子考第一就飘，下次回原形还怪运气
  • 运动员状态差请私教，回升就记私教功劳
  • 追高被套，坚信"这次不一样"
  • 把短期随机波动当成长期趋势
例：孩子考砸你一顿骂，下次好了，你以为是骂的功劳——其实只是回归。
✓

尊重回归

Respect the Pull
  • 看到极端数据，先问：真突破还是暂时偏离？
  • 投资在低于历史均值时布局，高于时收手
  • 对自己 / 孩子的起落少过度反应
  • 用"中枢"而非"峰值"做长期判断
例：巴菲特低买高卖——低于内在价值时买，高于时卖，借回归赚确定性。

四个关键认知

Key Insights
01

引力法则

极端值必被拉回价值中枢，就像地球引力把抛起的石头拽回地面——涨太多必跌，跌太多必涨。

02

必然性

股票不会永远涨也不会永远跌。短期看像广场上随机漫步的人，长期必被"引力"拉回中枢。

03

不对称性

涨 1000 点用 30 天，跌回来幅度和速度都随机，别幻想对称——回落可能更狠也更慢。

04

错误归因陷阱

把"回归"当成自己努力或失误的功劳，是人性最常见的自作多情，也是管理的最大盲区。

例：请私教后成绩回升，真相只是状态回归平均，而非私教点石成金。
05

康波长周期

人生发财靠 50–60 年康波周期，西格尔常量显示美股两百年实际年化约 7%——低买高卖才是王道。

识别信号 → 应对

Signal to Response

识别极端

看到暴涨 / 暴跌、超常业绩，先标记：这是被"引力"拉偏的极端值。

判断性质

是真突破，还是暂时偏离均值？多数"这次不一样"只是安慰剂。

投资布局

低于历史均值时布局，高于时收手，借回归赚确定性而非赌趋势。

克制反应

对孩子 / 自己的起落少过度干预，别在极端点做终身决定。

长期锚定

用中枢而非峰值做人生判断，把情绪从短期波动里摘出来。

Put It to Work

先问性质

✓

画一条自己的"价值中枢"，别被短期波动带乱节奏。

✓

每次想说这句话，先当成陷阱检查一遍再开口。
