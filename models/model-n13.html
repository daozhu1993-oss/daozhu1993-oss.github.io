Thinking Model · No.120

复利效应

Compound Effect
利滚利不是发财咒语，是给时间以耐心的人，时间给他的回礼。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 复利是本金、利率与时间三者的乘积效应（F=P(1+i)ⁿ）：让每一份收益都成为下一份收益的本金。它最迷人的地方不在高回报，而在拉得够长——雪球的关键是湿雪与长坡，不是第一下的力道。
Origin
岛主思维模型合集思想源自金融学“利滚利”；素材引述爱因斯坦戏称其为“世界第八大奇迹”
Field
财富 · 认知 · 成长投资、学习、个人长期主义
Core Definition
把收益再投入，让“利”生“利”随时间呈指数级放大，而非线性累加
Thinking Layer
成长 · 决策长期主义 / 时间杠杆

误用 vs 善用

Bad vs Good
✕

把复利当暴富杠杆

Get-Rich-Quick Fantasy
  • 追高回报高风险的“捷径”，本金先被赔光
  • 频繁换赛道，雪球刚滚起来就推倒重来
  • 只盯着公式里的利率，忽略最关键的期数（时间）
  • 三天打鱼两天晒网，永远到不了临界点
例：想一夜暴富去加杠杆炒币，本金直接归零，连“利滚利”的入场券都弄丢了。
✓

把时间当放大器

Patience as Multiplier
  • 选“湿雪长坡”——基础宽、能长期深耕的领域
  • 把每天的小投入固定下来，雷打不动
  • 攒跨学科模型，让知识互相碰撞“生息”
  • 接受前期慢，熬过临界点才爆发（如巴菲特）
例：巴菲特27岁起年复利20.5%，50岁前像普通中产，之后财富呈爆炸式增长。

复利的三块雪球

Three Snowballs
01

财富复利：时间是放大器

高回报常伴高风险，匀速稳定的复利反而赢在最后。大多数人想一夜暴富，结果把本金赔光。

例：巴菲特99%的财富是50岁之后才赚到的。
02

知识复利：学科碰撞出利息

芒格说“争取每天比前一天聪明一点点”。每天进步1%，一年后是37.78倍；每天退1%，只剩0.025。

03

复利型人才：本金大·利率高·期数长

选基础宽、专业精的职业（本金大），养好职业素养（利率高），长期稳定深耕（期数长）。

04

临界点之前最容易放弃

复利最难的不是懂，是“坚持到临界点”之前就放弃了。马云那句话最狠：多数人死在明天晚上。

05

不进则退，没有静止档

世界没有“保持原样”这个选项。你每天的小投入或小荒废，都在指数曲线上悄悄加码或减码。

三步开启你的复利

Start Compounding

选定领域

找一条“湿雪长坡”——基础宽、能长期做、有复利的方向，别频繁换赛道。

固定投入

把每天的小投入雷打不动地固定下来：写点东西、读几页、练一门手艺。

跨学科积累

攒思维模型，让不同学科的知识互相碰撞、彼此“生息”，而非单线堆积。

熬过临界点

接受前期慢、回报薄，坚持到爆发点。岛主亲测：连续三年每天写，回头认知已翻几倍。

马上能用的事

Put It to Work
✓

立一个“滚雪球”flag

选一个能长期做、有复利的方向，写下来当作你的湿雪长坡。

✓

设每日最小投入

比如每天读10页或写200字，固定在某个时段，雷打不动。

✓

建跨学科书单

每月加一个陌生领域的模型，让知识互相碰撞生息。

✓

关掉一个“负复利”习惯

比如无目的刷手机两小时，它也在指数级消耗你。

✓

季度复盘

看哪个方向已越过临界点，把资源向它倾斜。
