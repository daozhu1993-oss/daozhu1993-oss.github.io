三重心智模型

Triadic Mind Model
智商是车速，理性是方向盘——聪明和理性，从来不是一回事。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 斯坦诺维奇发现人脑有三重心智：自主心智（本能）、算法心智（智力）、反省心智（理性监控）。聪明人未必理性，两者只微弱相关——开得再快，方向错了也白搭。真正的理性，来自“反省心智”在旁边冷眼踩刹车。
Origin
基思·斯坦诺维奇加拿大认知科学家斯坦诺维奇提出“三重心智”
Field
认知科学 · 决策 · 理性心理学 / 行为经济学
Core Definition
把心智拆成自主·算法·反省三层理性来自“反省心智”的监控
Thinking Layer
认知 · 决策元认知 / 理性训练

误用 vs 善用

Bad vs Good
✕

以为高智商就理性

Smart ≠ Rational
  • 用高智商替代理性判断（门萨半数信占星）
  • 全靠直觉“自动驾驶”，不启动反省
  • 被“认知吝啬鬼”支配，能不思考就不思考
  • 中“污化程序”：庞氏骗局、一夜暴富的毒
例：清华教授被骗1760万、博士给P2P买单——高智商低理性翻车现场。
✓

给脑子装监控

Install the Monitor
  • 决策前问“我的目标到底是什么”，让反省心智上线
  • 遇大事强制脑中模拟几种后果（去耦推演）
  • 补概率与科学思维，抵制污化程序
  • 警惕属性替换、易得性偏差、过度自信
例：昆山龙哥若先在脑中“去耦”模拟（超不过去会撞车、对方可能反杀），结局或许不同。

三重心智拆解

Three Minds
01

自主心智：本能直觉

大脑边防兵。看见喜欢的人心动、骑车成条件反射，都是进化或长期训练内化的“自动驾驶”。

02

算法心智：我们说的“智力”

分流体智力（20岁达峰后下滑）和晶体智力（靠积累、一生发展）。传统智商测的只是它。

例：聪明人只有被告知怎么做，才表现优异。
03

反省心智：理性的摄像头

在旁边冷眼旁观，监控目标与信念，适时给算法心智踩刹车、换路线。

04

聪明≠理性

两者微弱相关。智商高不代表方向对，理性才是那个握方向盘的。

05

认知吝啬鬼

大脑能不思考就不思考。想变理性，得主动“装好程序”、避开先天缺陷。

三步练理性

Train Rationality

触发信号

做决定前先问“我的目标到底是什么”，把反省心智从待机叫醒。

启动反省

遇大事强制在脑中模拟几种后果（去耦推演）再行动。

补好程序

学概率与科学思维，抵制“污化程序”的毒。

自查偏差

定期核对是否掉进过度自信、易得性偏差等陷阱。
