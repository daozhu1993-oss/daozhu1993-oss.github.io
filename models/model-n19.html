耗散结构理论

Dissipative Structure Model
封闭系统走向死寂，开放系统才能从混乱跃迁到有序。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 一个远离平衡态的开放系统，只要持续与外界交换物质能量、越过临界点，就会从混沌中自发长出秩序——活水不腐，死水必凉。
Origin
普利高津（Ilya Prigogine）1977 年诺贝尔化学奖得主，以"耗散结构"统一物理界与生物界（素材原话）
Field
系统科学 / 热力学物理、生物与组织演化的统一框架
Core Definition
开放系统在远离平衡态时，靠"负熵流"抵消混乱、越过临界点后自发涌现有序结构。一句话：活水靠流动存活，死水靠静止走向寂灭
Thinking Layer
系统 / 认知看连接、看演化的上帝视角

误用 vs 善用

Bad vs Good
✕

关门死守，等系统自己好

Closed & Balanced
  • 把团队或自己关起来，信息不进不出，熵增至死寂
  • 追求"一团和气"的平衡态，墨滴扩散到均匀就是混乱
  • 把小涨落当噪音压制，错失跃迁的契机
  • 靠单一"负熵流"（如只吃药）幻想康复，忽略系统活力
例：封闭的公司从不引进外部人才，三年后流程僵化、创新归零。
✓

主动开放，在临界点附近跃迁

Open & Far-from-Equilibrium
  • 持续引进外部人才、信息、资金，注入"负熵流"
  • 保持"非平衡"张力，制造适度冲突与碰撞
  • 容忍小涨落，把它当新秩序的产房而非噪音
  • 推动系统越过临界点，让有序自发涌现
例：组织定期引入外部视角与跨界工作坊，碰撞出新业务方向。

关键要点

Key Points
01

必须是开放系统：关门必死

孤立系统熵只增不减，最终均匀死寂；开放系统引进负熵流，才能抵消混乱、长出秩序。

例：人体就是典型——吃进能量、排出废物，才活着。
02

远离平衡态：非平衡是有序之源

一滴墨在水里会扩散到均匀（平衡＝混乱）；只有被推到远离平衡的地方，系统才可能跃迁到有序。

03

非线性＋涨落：小扰动被放大

临界点附近，一个微小波动可能驱动系统跳进全新结构，就像知识碰撞越过阈值，突然涌现新思想。

04

负熵流不是万能药

光靠"药物"这类单一负熵流不够，还得让机体远离平衡、有活力，通过涨落冲过临界点才康复。

05

组织学习同理

封闭熵增致死，开放碰撞才创新——死读书没用，碰撞出新思想的组织才厉害。

借力四步

How to Apply

主动开放

多引进外部人才、信息、资金，别关起门来——先让负熵流进来。

制造非平衡张力

保持适度冲突与碰撞，别求一团和气，把系统推到远离平衡态。

容忍小涨落

把异常波动当新秩序的产房，而非噪音；别一有扰动就压下去。

推动越过临界点

持续注入负熵、制造碰撞，等有序在临界点附近自发涌现。

马上能用的事

Put It to Work
✓

给团队开一扇窗

本月引入一位外部视角（顾问或跨界朋友）做一次分享，注入负熵流。

✓

制造一点"不舒服"

在例会上专门安排反对意见环节，别求一团和气，保持非平衡张力。

✓

记录小涨落

把团队里的异常苗头记下来，月底看它是不是新方向的产房。

✓

检查自己是否"关门"

列出最近三个月没碰过的新信息源，挑两个补上，别困在平衡态。

✓

推动一个临界点

选一件卡住的事，持续小投入直到越过阈值，等有序自发冒出来。
