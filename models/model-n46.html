批判性思维

Critical Thinking
不是挑刺抬杠，是对自己和别人的想法，做严谨审慎的审视。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 批判性思维不是抬杠，而是对自己和他人的观点做多维度、有标准的审视——杜威管它叫"反思"，哈佛把它当核心课。
Origin
杜威"反思性思维" / 哈佛核心课岛主解读 / 思维模型合集
Field
思维训练 / 理性信息甄别、决策、沟通
Core Definition
对自己和他人的想法，做严谨、审慎、多维度的审视，而非盲从或抬杠。先问证据与逻辑
Thinking Layer
认知理性筛选信息

给自己装过滤器

Built-in Filter
  • 追问证据和推理站不站得住
  • 强制读一个相反观点
  • 常问"我凭事实还是情绪"
  • 经得起九把尺子检验
例：先看门道——作者目的、假设、证据是否站得住，再做判断。

关键要点

Key Points
01

思维有八件套

目的、问题、信息、假设、概念、推理、结果、意义——想不清，往往是这八样里缺了哪样。

例：看《包青天》也拆——目的是扬正义、假设是法律面前人人平等。
02

用九把尺子量

清晰、准确、精确、相关、深度、广度、逻辑、重要、公正。好观点经得起量，歪理一量就露怯。

例：广告说"百分百纯果汁"其实加了添加剂——不准。
03

养成七种特质

谦虚、勇气、换位、正直、坚毅、推理信心、自主。批判性思维先是态度，才是技术。

例：敢质疑权威、愿承认"我也可能错"。
04

先问证据再下结论

不被情绪带着跑，让证据和逻辑替你做主，而不是让热闹替你做主。

应用流程

How to Apply

停

遇到爆款结论先按下，别顺着情绪滑走。

问

作者的证据和推理站得住吗？用九把尺子量一量。

反

强制找一个相反观点，认真读一遍，别只听一面。

省

常问自己：这次，是凭事实还是凭情绪下的判断？
