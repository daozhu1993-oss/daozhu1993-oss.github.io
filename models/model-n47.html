禀赋效应

Endowment Effect
东西一旦到了你手里，你对它的估价就悄悄翻了倍。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 行为经济学家塞勒发现，一旦东西归你，你对它的估价立马翻倍——卖价能是买价的两倍，因为"失去"的痛远胜"得到"的乐。
Origin
塞勒 / 行为经济学（1980）岛主解读 / 思维模型合集
Field
行为经济学 / 消费心理交易、投资、断舍离
Core Definition
一旦拥有某物，人对它的主观估价显著高于未拥有时，卖价常为买价两倍。拥有即加价
Thinking Layer
认知 / 决策看穿自己的"感情价"

误用 vs 善用

Bad vs Good
✕

被"我家的"绑架

Held Hostage by Mine
  • 卖房觉得值一个亿，买家只肯出零头
  • 免费试用一个月后舍不得退
  • 股票跌了还找理由不卖
  • 被商家"以旧换新"套住
例：塞勒实验里，拿到咖啡杯的人要价中位数 5.25，没拿的只出 2.25–2.75——同一只杯子，位置不同价值差两倍。
✓

划清"喜欢"和"值钱"

Like vs Worth
  • 卖东西找第三方估价
  • 持股设机械止损线
  • 遇免费试用先提醒自己会舍不得
  • 区分"喜欢"和"值钱"
例：阿龙拆迁后急着还债，160 万就成交，没被效应绑架。

关键要点

Key Points
01

损失规避作怪

失去物品的痛，是得到同物快乐的 2.5 倍。卖家把卖房当"损失"自然抬价，买家把付钱当"损失"自然压价。

02

查询理论

买卖双方问自己的问题顺序不同，记忆提取的属性就不同，估价自然分歧。卖家先想"我能不卖吗"，买家先想"我能不买吗"。

03

情绪＋动机

开心时更惜物，急着还债时反而便宜甩。情绪和处境直接改写你的"底价"。

例：阿龙急着还债，160 万就成交。
04

物品属性

越稀缺、越个性化越容易高估；纯市场流通品（商店买鞋）反而不触发。免费试用、以旧换新，全是商家用它套你。

应用流程

How to Apply

识别

我对这东西的估价，是不是因为我"拥有"而偏高了？

验值

找第三方或市场价对照，别信自己的"感情价"。

止损

持股设机械止损线，别让"喜欢"替你扛跌。

警惕

免费试用、以旧换新前先打预防针：用久了就会舍不得退。
