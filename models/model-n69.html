Thinking Model · No.176

证伪思维

Falsificationism
一个理论只要找得到反例推翻它，才算站得住脚；永远正确的，多半是幻觉。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 波普尔一锤定音：科学和非科学的分界线，不是「能被证明」，而是「能被证伪」。知识不是真理，只是暂时更接近真理的猜想。
Origin
卡尔·波普尔（Karl Popper）1934 年《科学发现的逻辑》提出证伪主义
Field
科学哲学 · 决策 · 批判思维投资、战略、日常判断都该用得上
Core Definition
用「能否被反例推翻」来检验一个理论是否成立——可被证伪才是科学的，圆不回来的就是玄学。知识是「猜想—反驳」滚雪球逼近真理
Thinking Layer
认知 · 决策先质疑，再相信

误用 vs 善用

Bad vs Good
✕

把「验证过几次对」当真理

Confirmation Bias
  • 把「我验证过几次都对」当成「这就是真理」
  • 只收集支持自己的证据，反例一律无视
  • 含糊其辞啥都沾边，永远挑不出错
  • 笃信占星、鸡汤式断言，圆不回来也认
例：占星术有 40% 美国人坚信是科学，科学家却说它不是——因为它说啥都对、没法被推翻。
✓

主动找反例去推翻它

Falsification
  • 问「什么情况能证明我错」，敢这么问才算真信
  • 做决策前列一张反方清单，别只攒支持证据
  • 证伪度越高越值钱：覆盖广、说得精确才进步
  • 试错四步循环：问题→猜想→消除错误→新问题
例：任正非专门开「证伪会」，让大家证明战略是错的，提前堵盲点。

关键要点

Key Points
01

可证伪性是门槛

「所有天鹅都是白的」能被一只黑天鹅推翻，所以是科学命题；「所有天鹅都是天使化身」没法验证，就不是。

02

知识是暂时更接近真理的猜想

没有永恒真理，只有目前还没被推翻、暂时更接近真相的猜想。别把暂时的当绝对的。

03

证伪度越高越值钱

一个理论覆盖广、说得精确（比如万有引力定律），反而更容易被挑错，也越进步。含糊其辞最没用。

例：地心说被日心说推翻，日心说又被「太阳也不是中心」推翻。
04

试错四步循环

问题 P1 → 大胆猜想 TT → 消除错误 EE → 新问题 P2。科学就这么滚雪球往前走。

05

反过来想

芒格的「反过来想，总是反过来想」正是这味——与其证明自己对，不如先证明自己错。

例：芒格说：今年没颠覆掉去年喜欢的想法，说明你没进步。

识别信号 → 应对

Signal to Response

抓「笃定」

下次你坚信一个结论，先把它标记出来——越笃定，越该被检验。

列反方清单

主动找「什么情况能证明我错」，别只收集支持自己的证据。

做证伪实验

用最小成本试一次推翻它：做一次小测试，看反例会不会出现。

把犯错当迭代

错得起才创新得起。被推翻不是丢脸，是认知又往前挪了一步。

模型金句

In a Nutshell

能被推翻的，才配叫认知；永远正确的，多半是幻觉。

岛主（化用波普尔）

今年没颠覆掉去年喜欢的想法，说明你没进步。

查理·芒格

科学和非科学的分界线，不是能被证明，而是能被证伪。

卡尔·波普尔

马上能用的事

Put It to Work
✓

给一个笃定观点列反方清单

挑一个你坚信的结论，写下至少 3 条能推翻它的可能情形。

✓

决策前先问「什么能证明我错」

做重要决定前，强制列一张反方清单，和正方证据摆一起看。

✓

小成本做一次证伪实验

对一个假设，设计一次最小代价的测试，主动找反例。

✓

把年度错误当迭代记录

年底翻出今年被推翻的想法，写下你因此前进了多少。

✓

遇绝对化断言先查反例

听到「永远」「绝对」「所有人都」这类话，先想：哪里能找到反例？

When to Use
