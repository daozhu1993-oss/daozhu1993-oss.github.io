Thinking Model · No.178

逻辑思维

Logical Thinking
逻辑不是聪明，是让别人无法轻易夺走你判断的底盘。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 爱因斯坦点破：西方科学靠形式逻辑加实验找因果。逻辑思维，就是现代人讲清道理、不被忽悠的底盘——先框清概念，再补上推理链。
Origin
岛主解读 / 思维模型合集素材提及爱因斯坦、罗素等人对形式逻辑的论述，未单列单一提出者
Field
沟通 · 独立思考 · 批判性思维写作、辩论、做决策天天用得上
Core Definition
用清晰的概念、严谨的推理（归纳/演绎/类比）和三大规律（同一/矛盾/排中），把道理讲清楚、把谬误揪出来。概念是细胞，推理是引擎，规律守底线
Thinking Layer
认知 · 决策先讲理，再不被带节奏

误用 vs 善用

Bad vs Good
✕

被带节奏的「逻辑漏风」

Fuzzy Logic
  • 概念含糊，各说各话还以为聊的是一回事
  • 抛观点不留「因为…所以…」，靠感觉站队
  • 偷换概念、自相矛盾、模棱两可全中招
  • 听到绝对化结论不查反例，容易被忽悠
例：有人说「没人进过这屋，进去的人没出来」——同一律矛盾律一照，立马露馅。
✓

有底盘的「讲理通路」

Sound Reasoning
  • 开口前先定义关键词，框清内涵与外延
  • 抛观点补上推理链：因为…所以…
  • 三大规律守底线：不偷换、不矛盾、不模棱两可
  • 听到绝对化结论先查反例和偷换
例：谈「商品」前先说清：为交换而生产的劳动产品——概念清楚了，后面才不飘。

关键要点

Key Points
01

概念是知识的细胞

先把「内涵（它是什么）」和「外延（包括哪些）」框清。含糊的概念，讲啥都飘。

02

推理是核心引擎

归纳从特殊到一般，演绎从一般到特殊，类比靠相似性迁移。三种打法各有适用场景。

03

归纳的坑要当心

罗素那只火鸡用 100 天观察「证明」主人每天喂它，结果节日成了烤鸡——样本再大也概括不了未来。

例：靠经验下结论最容易翻车，岛主亲测。
04

三大规律守底线

同一律：苹果就是苹果，别偷换；矛盾律：不能既说 A 又说非 A；排中律：禁烟还是不禁，别模棱两可。

05

类比易以偏概全

看锯齿草发明锯子很妙，但类比靠相似迁移，容易「以偏概全」。迁移时要先确认相似点站得住。

应用流程

How to Apply

定义关键词

开口前先框清概念的内涵与外延，避免各说各话。

补推理链

抛观点时补上「因为…所以…」，让听的人能顺着你的逻辑走。

查三大规律

自查有没有偷换概念、自相矛盾、模棱两可——这三条是底线。

验反例

听到绝对化结论，先查它有没有反例、是不是偷换概念，再决定信不信。

马上能用的事

Put It to Work
✓

开口前先定义关键词

写一段话或开会前，先列清核心词的内涵与外延，避免各说各话。

✓

抛观点补「因为所以」

下次发看法，强制在观点后补一句推理链，让逻辑露出来。

✓

听到绝对化先查反例

遇到「所有」「永远」「肯定」，先想一个反例，再决定信几分。

✓

写一段话自查三大规律

挑自己最近一段论证，检查是否偷换、矛盾、模棱两可。

✓

读文章标出偷换概念处

读一篇观点文，用笔标出它偷换或含糊的地方，练出火眼金睛。
