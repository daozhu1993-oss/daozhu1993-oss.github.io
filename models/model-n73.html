OKR 目标管理

Objectives & Key Results
不考核、只对齐——让全队朝同一座山头冲；KPI 管住人，OKR 点燃人。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： OKR 把目标拆成「想去哪」(O) 和「怎么算到」(KR)，不做考核、全员公开，用对齐代替管控。
Origin
约翰·杜尔 · 1999 年引入 Google风靡 Google、Twitter 等科技公司的目标管理法
Field
目标管理 / 团队管理适用于个人与组织对齐方向
Core Definition
O 定方向、KR 定进度，用公开对齐代替 KPI 式考核的目标管理法。完成 60%–70% 即算健康
Thinking Layer
战略 / 执行上接愿景，下接每周动作

误用 vs 善用

Bad vs Good
✕

把 OKR 当 KPI 用

OKR as KPI
  • 写满数字指标，每个都硬扣钱，吓得人只挑安全的写。
  • 季度末打分排名，没达标就扣奖金，变成「数字民工」。
  • 目标不公开，部门各写各的，全公司对不齐。
  • 为了好看，把 O 也量化成冷冰冰的百分比。
例：某公司把 OKR 和绩效强绑定，员工只敢定低到肯定能完成的 KR，年底全员及格但创新归零。
✓

用对齐点燃自驱

Align to Ignite
  • O 写一句燃的话，不堆数字；KR 才用可衡量结果。
  • 打分只为复盘「为什么没到」，不扣钱。
  • 全员公开透明，周一盯责任、周五庆祝小胜。
  • 完成 60%–70% 视作健康，留出挑战空间。
例：Google 从搜索小公司长成巨无霸，靠全员公开 OKR + 季度复盘；西尔斯用了 OKR 的员工提拔概率高 3.5 倍。

关键要点

Key Points
01

O 是定性的「想去哪」

一句鼓舞人心的话，比如「提升用户服务体验」，不堆数字、不写百分比。

02

KR 是定量的「怎么算到了」

通常 2–3 个关键结果，用数字衡量，但数字不是拿来扣钱的。

03

三个特征最要命

不做考核、忘记失败分数、全员公开透明——KPI 是「必须达标」，OKR 是「我挑战」。

04

打分不为考核，只为复盘

季度末看「为什么没到 60%」，把复盘当燃料，而不是当判决书。

三步用起来

How to Use

分清 O 与 KR

O 是想去哪的方向感，KR 是到了没的尺子；先把二者在脑子里拆干净。

季度初写 1 个 O + 2–3 个 KR

O 要燃、KR 要可衡量，写完像立个 flag，让团队看得见山头。

全员公开 + 周节奏

周一盯责任、周五庆祝小胜利，把「对齐」变成每周呼吸。

季度末复盘不考核

打分只为问「为什么没到」，不打分扣钱，把复盘当下一季的燃料。

Put It to Work
✓

把一个 KPI 改成「挑战型 O」

明年的某个「必须完成」，试着换成「我想挑战」的燃向目标。

✓

给团队写 1 个 O + 2–3 个 KR

下周一对齐，先把山头指清楚，再谈怎么走。

✓

设「周五小胜利」节奏

每周五花 5 分钟庆祝一个 KR 进展，让对齐有温度。

✓

季度末只复盘不扣钱

问「为什么没到 60%」，把答案当成下一季的弹药。

✓

把一个野心放进公开 OKR

藏着的野心最易烂尾，公开了反而有人一起推你。
