Thinking Model · No.181

世界咖啡

World Café
像喝咖啡那样围坐、换桌、异花授粉，把一群人的盲点搅成集体智慧。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 世界咖啡用「多轮换桌 + 跨界混搭」的轻松对话，让不同大脑互相补盲点，搅出单个人想不出的解法。
Origin
朱尼塔·布朗 & 戴维·伊萨克斯 · 1990 年代被彼得·圣吉称为「进入集体创造最可靠的方式」
Field
团队共创 / 集体智慧适用于会议、创新、共识场景
Core Definition
4–6 人一桌、多轮换桌把观点接力，用轻松对话激发集体智慧的方法。桌长留任，其余换桌带精华
Thinking Layer
协作 / 创新把个人盲点交给群体补丁

误用 vs 善用

Bad vs Good
✕

开成念稿大会

Lecture Meeting
  • 一个人念 PPT，其他人刷手机，结论老重复。
  • 固定座位、固定圈子，观点在原地打转。
  • 谁职位高谁说了算，平等感缺失，没人敢说真话。
  • 拿有标准答案的问题来讨论，纯属浪费时间。
例：某公司周会全员坐原位听总监念稿两小时，散会没人记得讨论了啥，下周的会照旧。
✓

让大脑异花授粉

Cross-Pollinate
  • 4–6 人一桌，第一轮深聊，之后除桌长外全员换桌。
  • 把上一桌的精华当接力棒带去下一桌，越搅越深。
  • 人人平等坐一条凳，卸掉防卫，盲点被补成补丁。
  • 最后全员汇谈，把散点连成一张解决图。
例：英特尔研发芯片时请社会学家、人类学家泡进世界咖啡，竟从人的需求出发捣鼓出无线上网。

关键要点

Key Points
01

一台集体智慧搅拌机

像三维网，越搅越深、越连越广，跨界大脑互相授粉。

02

五大价值

创新法宝、解题利器、团队共识、共同学习、情感交流——一个方法全包。

03

人人平等是前提

地位高不高都坐一条凳，卸掉防卫，真话才出得来。

04

真问题才能搅出真东西

别拿念稿能解决的事来开，问题得没有现成答案。

三步用起来

How to Use

挑「真问题」

选一个没有现成答案的难题，别拿来念稿、别预设立场。

4–6 人一桌，限时深聊

第一轮只管往深了聊，让每桌先长出自己的观点。

多轮换桌，观点接力

除桌长外全员换桌，每人带「上桌一句话精华」去下一桌。

全员汇谈，连点成图

最后把散点汇成一张解决图，让集体智慧看得见。

Put It to Work
✓

开会先筛「有标准答案吗」

有答案的别开咖啡，把世界咖啡留给真难题。

✓

按 4–6 人分组，设桌长

桌长留任串场，其余人负责把观点带去下一桌。

✓

第一轮只深聊别给结论

让每桌先长出自家观点，别急着收敛。

✓

换桌带「一句话精华」

提醒每人把上一桌最亮的点，当接力棒带过去。

✓

收尾做全员汇谈

把散点画成一张图，让集体智慧落进方案。
