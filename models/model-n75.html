定位理论

Positioning Theory
竞争的终局不在工厂、不在货架，而在顾客心智里那格小柜子。

一句话总览： 定位理论说，产品过剩时代，占住顾客心智里某个品类的梯子层，比占住货架更致命。
Origin
特劳特 · 1969 年提出被誉为「对美国营销影响最大的观念」
Field
品牌营销 / 战略适用于产品、个人、品牌占位
Core Definition
在顾客心智中为品牌找到差异化位置，抢占品类梯子某一层的理论。占住认知比占住货架更值钱
Thinking Layer
战略 / 认知上接产品，下接顾客脑子

失焦式扩张

Brand Dilution
  • 格力做手机、海尔做手机没人信——专业的人不专业了。
  • 什么功能都塞，顾客记不住你是谁。
  • 正面硬刚品类老大，用劣势撞人优势。
  • 以为「酒香不怕巷子深」，好产品不占位也卖得动。
例：某企业本是空调专家，非要出手机，消费者一句话「你认真做空调不行吗」，新品类草草收场。
✓

钉进心智格子

Own the Ladder
  • 看清顾客心智里你的品类梯子，谁占着头层。
  • 语言钉 + 视觉锤：一句话 + 一眼认出的符号。
  • 要么抢先占空位，要么和老大对着干。
  • 守住专业，不轻易失焦跨到自己没信用的品类。
例：七喜说「非可乐」，硬在可口与百事间撕出老三；特斯拉开「电动汽车」新格子，成了品类代表。

关键要点

Key Points
01

心智像一格格小柜子

每个品类有把梯子，你不上某一层，顾客就看不见你。沃尔沃=安全、宝马=驾驶乐趣。

02

心智五条脾气

容量有限、厌恶混乱、缺乏安全感、不愿改变、容易失焦。

03

语言钉 + 视觉锤

「怕上火喝王老吉」（钉）+ 红罐（锤），双重钉进脑子。

04

抢位或对立

七喜「非可乐」对立定位；瓜子二手车砸 10 亿饱和攻击占头部。

05

顾客认的「事实」更值钱

认知里的位置，比物理参数更能决定购买。

三步用起来

How to Use

看清品类梯子

先搞明白顾客心智里你的品类梯子长啥样、谁占着头层。

找一句钉 + 一个视觉锤

写一句能钉进脑的话，配一个一眼认出的视觉符号（颜色/形状）。

抢空位或对立定位

要么抢先占空位，要么和老大对着干，别正面硬刚。

饱和攻击守住位置

把资源集中砸一个认知点，反复强化，别撒胡椒面。

Put It to Work
✓

画出你的心智梯子

✓

写一句能钉进脑的话

✓

找一个一眼认出的符号

颜色、形状、包装都行，让视觉替你喊名字。

✓

决定抢空位还是对着干

别正面硬刚老大，要么新格子、要么撕个老三出来。

✓

别撒胡椒面，把预算压在单一心智位置上反复打。
