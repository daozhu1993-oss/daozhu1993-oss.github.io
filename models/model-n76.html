库伯学习圈

Experiential Learning Cycle
完整的学习不是一条线，而是具体经验→反思→抽象→实践，转不停的圈。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 库伯学习圈把学习拆成四步闭环：先干、再想、再总结、再拿去用，缺一环都等于白学。
Origin
大卫·库伯 · 1984 年提出奠基自杜威、勒温、皮亚杰的学习观
Field
学习科学 / 教育适用于个人成长与培训
Core Definition
从具体经验出发，经反思、抽象到主动实践的四阶段循环模型。停在第二层就回去，等于白学
Thinking Layer
成长 / 认知把知识变成本能

误用 vs 善用

Bad vs Good
✕

只走一半的学习

Half-Learned
  • 只听课看书，从不真去干，经验层都没进。
  • 干完就跑，不回头想「哪儿卡了」，停在第二层。
  • 从不总结规律，下次遇到还是从头懵。
  • 学完不应用，知识躺在收藏夹里吃灰。
例：有人报了一堆写作课，从不复盘哪句有效，三年过去文笔和第一天一样——卡在第二层就回了家。
✓

转完四步才真会

Complete the Loop
  • 具体经验：先跳进去干，看教程是别人的经验。
  • 反思性观察：停下来想哪儿顺、哪儿卡。
  • 抽象概念化：把碎片拼成自己的方法地图。
  • 主动实践：拿去用、再去撞，圈又转起来。
例：岛主学写作，每篇完事都问「这次哪句有效」，才慢慢长出方法——这就是第三层在起作用。

关键要点

Key Points
01

完整学习是个圈，不是条线

四层转起来才闭环，缺哪层都算没学完。

02

具体经验是第一脚

不下水学不会游泳，看教程是别人的经验。

03

反思把感觉捞成碎片

干完别跑，把模糊的感觉捞成清楚的碎片。

04

抽象把碎片拼成地图

总结出可复用原则，下次直接上方法而非从头懵。

补上缺的那一步

Close the Gap

写下「我实际做了什么」

下次学任何东西，先如实记录行动，别只记别人的干货。

花 5 分钟复盘「哪里不对劲」

干完别急着跑，把卡点和顺处捞成清楚的碎片。

提炼一条可复用原则

把碎片拼成自己的方法地图，写进笔记。

下周一就用一次

拿去真问题里撞，一用就发现新问题，圈又转起来。
