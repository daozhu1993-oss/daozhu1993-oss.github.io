魏朱商业模式

Stakeholder Transaction Structure Model
商业模式不是产品之争，是利益相关者之间怎么分钱、分权、分活的交易结构。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 魏朱把商业模式定义为「利益相关者的交易结构」——同样的技术，拼法不同，有人翻身有人沉底。
Origin
魏炜（北大）· 朱武祥（清华）提出脱胎于德鲁克「企业竞争是商业模式之争」
Field
商业战略 / 创业适用于设计、诊断生意
Core Definition
企业最大化价值而构建的利益相关者交易结构，由六块拼图组成。定位·业务系统·盈利·资源·现金流·企业价值
Thinking Layer
战略 / 商业上接技术，下接生态共赢

误用 vs 善用

Bad vs Good
✕

只盯客户和对手

Narrow View
  • 眼里只有客户和竞品，忘了供应商、司机、渠道也是玩家。
  • 死磕产品功能，不琢磨钱从哪进哪出。
  • 用垄断吸干生态，短期爽长期崩。
  • 复制别人模式，不问自己利益相关方能不能这么玩。
例：某公司照抄对手的低价模式，却没重算现金流结构，烧钱补贴一年就资金链断裂。
✓

重拼交易结构

Rebuild the Deal
  • 列出所有利益相关方，不止客户和对手。
  • 问彼此有什么价值可交换，重切产权和收支。
  • 用三把尺检验：高价值、低成本、能共享。
  • 让多方共赢，而不是自己独吞。
例：四川航空让卖车行、司机、旅行社、航司、乘客五方全赢——把交易结构重新拼了一遍。

关键要点

Key Points
01

六块拼图

定位、业务系统、盈利模式、关键资源能力、现金流结构、企业价值。

02

好模式三把尺

高价值创造、低价值耗散、价值共享——垄断不算好模式。

03

利益相关者视角

滴滴没车却成最大出行公司，赢在交易结构而非资产。

04

模式之争 > 产品之争

德鲁克说，企业竞争不是产品之争，是商业模式之争。

05

重切收支与产权

同样生意，换种分钱分权方式，就能多方共赢。

三步用起来

How to Use

列出利益相关方

别只盯客户和对手，把供应商、渠道、用户都摆上桌。

找可交换价值，重切产权收支

问彼此「有什么价值可交换」，重新切分产权与收支方式。

用三把尺检验新结构

高价值、低成本、能共享——三条都过才算好模式。

小范围试点验证

先在一个环节试新结构，看现金流是否真的更健康。
