合作模型

Repeated-Game Cooperation Model
合作的基础不是信任，而是关系的持续性——还要长期打交道，合作就自然长出来。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 阿克塞尔罗德用《合作的进化》告诉我们：哪怕彼此戒备，只要还要长期打交道，合作就会自然长出来。
Origin
罗伯特·阿克塞尔罗德 · 《合作的进化》源自重复博弈与囚徒困境研究
Field
博弈论 / 人际关系适用于商业合作、团队
Core Definition
合作不靠预先信任，而靠关系持续性与「以牙还牙」规则自发维持的模型。单次博弈背叛理性，重复博弈合作涌现
Thinking Layer
认知 / 关系把名声当存折

误用 vs 善用

Bad vs Good
✕

一锤子买卖

One-Shot Deal
  • 一次性交易，知道你下次不来，能宰就宰。
  • 无底线宽容，背叛没代价，养出白眼狼。
  • 耍小聪明短期占便宜，名声存折透支。
  • 以为先信任就能换来合作，结果被坑。
例：景区摊贩对游客狠宰，因为算准你这辈子不会再来了——单次博弈里背叛最理性。
✓

把关系拉成长线

Play the Long Game
  • 把一次性变长线，让「下次还见」替你盯住对方。
  • 定清楚以牙还牙规则：合作给回报，背叛有代价。
  • 不嫉妒、不先叛、有回报、不耍小聪明。
  • 持续关系本身就是约束，名声成了隐形契约。
例：岛主把供应商合作频率拉高、验收做严，半年后对方从偷工减料变成最靠谱——关系持续，人品也进化。

关键要点

Key Points
01

单次博弈，背叛理性

囚徒困境里招供最划算，就像景区摊贩敢宰客。

02

重复博弈，以牙还牙最稳

开局信任，对方怎么对我，下一轮就怎么回敬。

03

持续关系即约束

办婚礼就是把长期接触公开，让背叛成本高到不敢动。

04

阿克塞尔罗德四真言

不嫉妒、不先叛、有回报、不耍小聪明。

三步不翻车

How to Use

变一次性为长线

能续约就别一锤子买卖，让「下次还见」替你盯住对方。

定以牙还牙规则

合作给回报，背叛有代价，白纸黑字写清楚。

不耍小聪明，护名声

对方不傻，短期占便宜长期必赔，名声存折别透支。

公开关系，放大约束

把长期接触公开化，让背叛成本更高、更不敢动。
