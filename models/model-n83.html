行动学习画布

Action Learning Canvas
开个会就把真问题解决了——4 到 8 人平等互助，六步画布把"聊天"变成"同屏地图"。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 行动学习画布把瑞文斯的"行动学习"画成一张同屏地图：提出难题、澄清、重构、创新方案、行动、反思，六步走完，让团队在互助中解决真问题。
Origin
瑞文斯（英国）行动学习之父提出，中国融合创新为"画布"
Field
组织发展 / 团队管理高效会议、人才发展、问题解决
Core Definition
用 AL=P+Q+A+R（程序知识+洞察提问+行动+反思）公式，让 4-8 人平等互助，在解决真实难题中成长画布让思考看得见、可同屏
Thinking Layer
成长 / 方法在战争中学习战争，借事修人

误用 vs 善用

Bad vs Good
✕

开会原地打转

Meetings That Spin
  • 各说各话，走出会议室还是没共识
  • 培训学完回到岗位还是老样子（学用脱节）
  • 只改行为不治本（单环学习）
  • 没有真难题，为了开会而开会
例：三小时会议吵完，问题原地打转，下周还是同一摊烂事。
✓

画布剥洋葱

Peel with the Canvas
  • 4-8 人平等互助，"没大没小没上没下"
  • 六步把聊天画成同屏地图，减少误解
  • 先重构"真问题"，再谈方案
  • 双环学习，反思行为背后的假设（治本）
例：华润置地 2001 年用行动学习，原以为是"物业效率低"，深挖才发现是"架构职责不合理"，效率从 18 个月缩到 11 个月。

三层看懂画布

Three Layers
01

第一层 · 本质是一张互助网

公式 AL=P+Q+A+R：程序知识 + 洞察性提问 + 行动 + 反思，4 到 8 人平等互助解决问题。

02

第二层 · 画布让思考可见

把聊天画成同屏地图——提出难题、澄清、重构、创新方案、行动、反思，六步走完不跑偏。

03

第三层 · 双环学习是根本解

单环只改行为（治标），双环反思行为背后的假设（治本）。岛主带团队时一问"真问题是啥"，答案常完全反转。

04

破解三大障碍

传统培训有技术性、适应性、制度性三障碍，行动学习用"真事真练"一一击破。

画布六步走

The Six Steps

提出难题

每人带一个真实、棘手的难题进会议室，不聊虚的。

澄清与重构

互相提问把问题澄清，再用画布重构出"真问题"。

创新方案

针对真问题头脑激荡，画布上并列候选方案不跑题。

行动与反思

当场定责任人下周落地一小步，事后双环反思假设是否站得住。
