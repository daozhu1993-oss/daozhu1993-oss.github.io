集体行动模型

Collective Action Model
公地悲剧不是人性本恶，而是制度没把"个人的账"算进"集体的盘"。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 集体行动的困境，本质是"我贡献了也分不到，不贡献也少不了"的算计——公地悲剧不是人性本恶，而是制度没把个人的账算进集体的盘。
Origin
斯科特·佩奇《模型思维》奥斯特罗姆以公共事务研究获诺贝尔经济学奖
Field
制度经济学 / 社会治理公共事务、团队合作、资源管理
Core Definition
当利益归个人、成本摊集体时，理性个体倾向搭便车或过度索取，导致公共品枯竭个体理性 vs 集体非理性
Thinking Layer
战略 / 制度从人性到规则的升维

误用 vs 善用

Bad vs Good
✕

把成本摊给无关的人

The Tragedy of the Commons
  • "少数服从多数"把垃圾点按在别人家门口
  • 公共品非排他，谁都想着"我不交也行"
  • 拥塞让共享变互相伤害，收益归零
  • 开采超过均衡线，资源会突然崩
例：小区垃圾分类点选址由物业全员投票，除点位附近没人反对，于是"多数"把垃圾点按在别人家门口——被伤的人根本不是自愿的。
✓

把边界划清、让用的人自己定规则

Let Users Write the Rules
  • 明确"谁用谁负责"，不把成本摊给无关的人
  • 让受影响的人参与定规则，公平感才出信任
  • 设分级惩罚，初犯轻罚、屡犯重罚
  • 政府认其合法性，使用者自己就能管好
例：沿海渔场由渔民自定"谁能捕鱼、用什么设备"，只要政府认这个规则的合法性，他们自己就能把资源管住。

困局的三个源头

Three Sources
01

非排他性养出搭便车

国防、空气、公园你我都能用，你少出一份力也照常享受。就像新农合，人人交一样的钱，但有人总想"我不交也行"。

02

拥塞让共享变互相伤害

国庆高速免费，结果大家都上路，堵到谁都动不了，收益直接归零。道路像一口锅，人满就糊。

03

可再生资源有临界点

长江十年禁渔前，过度捕捞让江豚快没了——开采超过均衡线，资源会突然崩。

奥斯特罗姆：她研究了一辈子公共事务，八条原则核心就一句——把边界划清、让用的人自己定规则。
04

被设计的"自愿"才有效

最低效的集体决策，是让被伤的人根本不是自愿的。规则若不管"个人的账"，再团结也白搭。

三步改善集体困局

Three Steps

划清责任边界

明确"谁用谁负责"，别让成本摊给无关的人，先把账算清楚。

让当事人定规则

让受影响的人参与定规则，公平感出来了，信任才跟得上。

设分级惩罚

初犯轻罚、屡犯重罚，别让老实人吃亏——制度要替好人撑腰。

✓

警惕"多数欺负少数"

✓

共享资源设开采上限，别等"突然崩"才后悔，临界点不等人。
