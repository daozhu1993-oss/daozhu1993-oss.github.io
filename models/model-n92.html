奥卡姆剃刀

Occam's Razor
如无必要，勿增实体——能少则少，简单才是最高级的复杂。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 当两个理论都能解释现象，选假设少的那把。奥卡姆的威廉留下的"剃刀"，把多余的假设、论证、枝蔓统统刮掉，只留最省事还能成事的版本。
Origin
奥卡姆的威廉（14 世纪）英格兰逻辑学家 · 经院哲学
Field
逻辑与决策也适用管理 · 生活 · 科学
Core Definition
能用更少假设解释现象的，就不要增加多余的实体 / 假设简单有效，而非简单了事
Thinking Layer
认知 / 决策化繁为简 · 抓本质

误用 vs 善用

Bad vs Good
✕

功能越堆越多，最后谁都看不懂

Complexity for Its Own Sake
  • 做方案功能越加越多，以为"多 = 好"，结果没人看得懂。
  • 听大师讲玄学，越听越迷糊，却说不出哪里不对。
  • 家里东西越买越乱，断舍离喊了一年还没动。
  • 用十个线索破案，看似严谨，实则把自己绕晕。
例：岛主曾囤 300G 资料以为"收藏即学会"，越囤越焦虑，反而一本没读。
✓

一把剃刀，刮掉多余

Shave Off the Unnecessary
  • 假设越少越接近真相：两个理论都能解释，选假设少的。
  • 简单才能灵活：信息传得快，组织转得动。
  • 生活里是断舍离：不是整理术，是放下执念。
  • 简单是最高级的复杂：E=MC² 几个字母概括整个宇宙。
例：乔布斯重返苹果，第一刀把 60 多条产品线砍到 4 条；韦尔奇砍掉 117 个事业部，通用才活过来。

关键要点

Key Points
01

假设越少越接近真相

两个理论都能解释现象，选假设少的那把。哥白尼一把剃刀把 84 个"均轮"减到 34 个，反而更对。

02

简单才能跑得灵活

企业里，简单的信息传得快、简单的组织转得动。复杂是运转的敌人，不是能力的证明。

03

生活里的断舍离

山下英子说"断绝不需要、舍弃多余、脱离执着"，东西少了，心才透亮。放下执念，比整理更紧要。

04

大脑是吝啬鬼

假设越多越耗能。能省则省，把认知带宽留给真正要命的问题，而不是枝蔓。

05

简单是高级的复杂

爱因斯坦质能公式几个字母，概括整个宇宙。极致的理解，往往长得最朴素。

应用流程

How to Apply

决策前先问"能不能再少一步"

对任何方案、流程、承诺先砍一刀。多出来的那一步，往往只是习惯，不是必需。

写方案先砍三分之一功能

看剩下的版本是否还成立，不成立再补。多数"必备"功能是自己吓自己。

每周清一次库存

物理的东西和信息（资料 / 收藏）一起清。反弹的代价，远小于囤积的内耗。

用"剃刀"审观点

听到玄学 / 复杂论证，先问：更简单的解释是什么？先把那把留下，再谈别的。

Put It to Work
✓

给方案动刀

明天把一份冗长文档 / 方案砍掉三分之一，看是否反而更清楚。

✓

清理一个数字角落

删掉收藏夹里八成"以后再看"的链接，它们从没被看过。

✓

遇争论先找简版

下次听人讲复杂道理，先问"最简单的解释是啥"，别被术语绕进去。

✓

设每周清空日

固定一天清物理和信息库存，防反弹，让大脑轻一点。

✓

少做"以防万一"的事

列出三件为"万一"多花的精力，能省就省，多数万一从没发生。
