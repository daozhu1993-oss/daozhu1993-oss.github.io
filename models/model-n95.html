Thinking Model · No.202

颠覆性创新

Disruptive Innovation
巨头不是被正面打败的，是被"低端边缘"一点点蚕食的。
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 克里斯坦森在《创新者的窘境》里点破：大公司像被地心引力拉着往上爬，脚下却空了。颠覆者先服务巨头看不上的"非消费者"，等技術一路打进主流，巨头才反应过来。
Origin
克里斯坦森（哈佛）《创新者的窘境》
Field
战略 / 创新商业 · 创业
Core Definition
从低端或新市场切入，用更简单便宜的方案，最终颠覆主流巨头错位生存，而非正面刚
Thinking Layer
战略 / 认知竞争 · 增长

误用 vs 善用

Bad vs Good
✕

在巨头的强项上死磕

Fighting Where Giants Win
  • 总想"比竞品功能多"，在巨头强项上正面刚。
  • 公司越做越大，主动放弃低端市场，留出真空。
  • 看不懂新玩家从哪冒出来，等反应过来已晚。
  • 技术更新慢还硬上，刚冒头就被灭（像小灵通）。
例：柯达发明数码相机却舍不得胶卷生意，反被别人颠覆；岛主早年"比功能多"被免费小团队抢走客户。
✓

从边缘长出新物种

Grow From the Edge
  • 错位生存：先服务巨头看不上的"非消费者"。
  • 利用巨头"东北角迁移"留下的真空。
  • 技术要能持续涨，一路打进主流才真颠覆。
  • 大公司建独立小团队，隔离旧流程旧价值观。
例：沃尔玛从折扣店、拼多多从五环外、抖音从"记录普通生活"起家；微信独立于 QQ 长成万亿新物种。低端切入成功概率 67%，正面刚仅 20%。

关键要点

Key Points
01

错位生存，不抢主流

颠覆者先服务巨头看不上的非消费者——便宜、方便就活下来。沃尔玛、拼多多、抖音都从边缘起家。

02

巨头"东北角迁移"留真空

大企业为利润向上爬，主动放弃低端市场。阿里从淘宝到天猫，给了拼多多缝隙；柯达舍不得胶卷，反被颠覆。

03

技术要能持续涨

小灵通便宜却更新慢，刚冒头就灭；真正颠覆的，是技术能一路打进主流。涨不上去，只是昙花一现。

04

建独立小团队

微信独立于 QQ，隔离旧流程旧价值观，才长成新物种。大公司不甩开包袱，新业务必被旧逻辑绞杀。

05

概率优势明显

克里斯坦森统计：从低端切入，新创成功概率 67%，正面刚只有 20%。选对战场，胜率翻倍。

应用流程

How to Apply

找巨头懒得做的边缘市场

别硬刚强项，盯"非消费者"和低价需求。巨头嫌弃的地方，恰恰是你活下来的缝隙。

用更简单便宜的方案立住

在边缘先把"活下来"这件事做扎实。单点戳心，比功能堆砌更有效。

建隔离的独立小团队

大公司要甩开旧流程旧价值观，给新物种空间。不隔离，就被旧 KPI 拖死。

留苗子，允许多点试错

留住"不赚钱但可能长大"的业务。今天的边角，也许就是明天的万亿新物种。

马上能用的事

Put It to Work
✓

画一张边缘地图

列出你行业里巨头懒得做的低端 / 新需求，那里藏着你的切入点。

✓

砍掉无效功能堆

别再"比竞品多"，先做一个单点戳心的产品，在边缘立住。

✓

给新业务独立编制

若在大公司，推动新项目脱离旧 KPI，别被旧流程绞杀。

✓

留一个试错预算

拨一小笔钱养"不赚钱但可能长大"的苗子，给未来留个口子。

✓

盯技术曲线

判断你的方案能不能持续涨、打进主流，涨不上去就趁早换道。
