SWOT

SWOT Analysis
别再列四象限了，关键是那四条交叉线  ·  韦里克  ·  美国管理学教授
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 美国管理学教授韦里克早说了——SWOT 的价值不在四个格子，而在格子之间的连线。多数人只填格子不画交叉线，等于买了地图不看路，白分析。
Origin
韦里克美国管理学教授，提出 SWOT 战略分析框架。
Field
战略分析 / 决策把内外部拧成一条能落手的路。
Core Definition
四格 + 四条交叉线S/W 内部家底，O/T 外部天气，连线成 SO·WO·ST·WT。
Thinking Layer
战略连接层不只画地图，更要画出从哪条线走。

误用 vs 善用

Bad vs Good
✕

只填格子

Fill Boxes, Skip Roads
  • 把 SWOT 当填空游戏，列完就扔
  • 四个格子填得满满，动手又拍脑袋
  • 只填格子不画交叉线，白分析
  • 仪式感代替了思考
例：像写完待办清单就心安理得刷手机，仪式感代替思考。
✓

画交叉线

Connect the Boxes, Walk the Road
  • 先填四格，再重点画四条交叉线
  • 挑一条最该打的组合，别四条都想打
  • 这周就为那条线排一个动作
  • 看清自己在哪条线，比填满格子有用十倍
例：本地生活团队用 SO 杠杆（地推铁军+下沉爆发）半年起量。

四条交叉线

The Four Cross-Lines
SO

杠杆 · 增长型

用优势撬机会，最爽的组合，顺着打就行，别犹豫。好钢得用在刀刃上。

例：地推铁军（S）+ 下沉爆发（O），直接打穿。
WO

扭转 · 补劣势

借机会补劣势，该招人招人该买买，别硬扛短板，借势补短板最省力。

例：小米早期没渠道（W），借换机潮（O）用互联网直销补齐。
ST

防御 · 筑墙

用优势顶威胁，筑墙护城河，用长板挡刀，以攻代守。

例：华为被制裁（T）后用自研芯片与鸿蒙（S）顶住。
WT

脆弱 · 躲或撤

劣势撞威胁，要么躲要么撤，别拿鸡蛋碰石头，认怂不丢人，硬撑才亏大。

例：小厂没渠道又遇巨头压境，乖乖躲进 niche 活成隐形冠军。

用起来四步

From Matrix to Move

老实填四个格子

先别跳过，缺一角交叉就歪。S/W 内部，O/T 外部，缺一不可。

重点画四条交叉线

挑一条最该打的组合，别四条都想打，贪多必乱。

这周就排一个动作

别停在 PPT 上落灰，为那条线排一个具体动作。

盯机会最大的那条

最该打的那条，往往不是你最擅长的，而是机会最大的。

模型金句

In a Nutshell

格子是地图，交叉线才是路；只看格子的人，永远在原地。

SWOT 的本质

看清自己在哪条交叉线，比填满四个格子有用十倍。

别只填格子

交叉线里最该打的那条，往往不是你最擅长的，而是机会最大的。
