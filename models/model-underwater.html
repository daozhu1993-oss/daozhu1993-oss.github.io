冰山胜任力模型

Competency Iceberg Model
岛主解读  ·  胜任力 / 冰山衍生  ·  源自 McClelland 1973
← 返回思维模型图鉴 ← 回岛主主页
一句话总览： 决定一个人能走多远的，不是水面上的 30% 显能，而是水下那看不见的 70%——性格、价值观与动机。简历帮你进门，冰山决定你能走多远。
Origin
戴维·麦克利兰David McClelland，1973 年提出胜任力理论，冰山为其经典隐喻。
Field
职业发展 / 人才评估 / 个人优势看人、看己、做 hiring 与晋升的底层框架。
Core Definition
显性 30% + 隐性 70%水上：知识·技能·通用能力；水下：特质·价值观·动机。
Thinking Layer
模型型水面技能 vs 水下特质/动机，决定长期发展。

只看简历 vs 看重冰山

Bad vs Good
✕

只看水上部分

Surface-Only Hiring
  • 用证书、学历、技能标签筛人
  • 忽略动机和价值观是否匹配
  • 招来"能力强但求安稳"的人
  • 遇到艰难项目，对方找借口推掉
例：A 员工技能强，但动机是"求安稳"，开拓新市场时敷衍执行。
✓

打捞水下冰山

Read the Iceberg
  • 看性格、价值观、动机是否匹配航道
  • 重用"冰山深厚"的人，长期更可靠
  • 让成就驱动的人主动请缨、全力以赴
  • 自己也持续增厚水下部分，建立壁垒
例：B 员工同样强，但动机是"成就驱动"，主动请缨、全力以赴。

能力冰山六要素

Six Layers
水上

知识 Knowledge

你学过什么？专业知识、行业认知。显性、易观察，是你的入场券。

水上

技能 Skill

你会用什么？软件操作、外语、编程。同样显性易培养，但替代性高。

水上

通用能力 Ability

你如何做事？沟通、协作、学习力。比知识技能更底层，仍是显性。

水下

性格特质 Trait

你是怎样的人？乐观、坚韧、正直。隐性、难改变，却持续驱动行为。

水下

价值观 Value

你认为什么对？崇尚合作、追求卓越。隐形方向盘，决定你往哪走。

水下

动机 Motive

你真正渴望什么？成就、权力、亲和驱动。发动机，最深处也最关键。

三步构建深层优势

Build Your Iceberg

自我探测

回顾最自豪/最挫败的经历，背后体现了你怎样的性格与看重什么；做什么事能让你废寝忘食。

对标校准

你的隐性特质，和当前岗位、团队、公司文化匹配吗？内心渴望自由却身处服从流程，会很痛苦。

刻意强化

承担有挑战的小任务锻炼韧性；面临选择时用原则做决定；拆大目标创造心流，滋养动机。

打捞并展示

别只盯证书，主动把"水下优势"（责任心、成就动机、坚韧）讲给对的人听，建立壁垒。

模型金句

In a Nutshell

显性能力决定你的起点，隐性素质决定你的顶点。

冰山法则

职业生涯前半程拼水上硬实力，后半程拼水下软实力与心之力。

长期主义

花点时间打捞你水下那座巨大的、独特的冰山——那才是你无法被轻易替代的真正资本。

岛主批注

马上能用的事

Put It to Work
✓

画自己的冰山

分两栏写下水上技能与水下特质/动机，看哪边更厚。

✓

问"什么让我废寝忘食"

找到自己的核心动机（成就/影响/归属），用它校准方向。

✓

检查人岗匹配

内心渴望创新自由，却身在强调服从的环境？那会持续内耗。

✓

招人别只看简历

多问动机与价值观，找"冰山深厚"的人，长期更可靠。

✓

用原则做选择

面临岔路时，有意识地用你的价值观做决定，强化它。

✓

主动讲出水下优势

面试/述职时，把责任心、成就动机说清楚，建立不可替代感。

什么时候该掏出它

When to Use
职业规划 招聘 / 晋升评估 自我优势梳理 团队搭建 判断人岗匹配 核心竞争力建设 任何"看懂一个人"的时刻
